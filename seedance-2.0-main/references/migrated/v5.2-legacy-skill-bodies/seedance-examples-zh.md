# seedance-examples-zh

Use this skill for Chinese examples and safe adaptations. Examples are templates, not permission to copy protected characters, branded ads, copyrighted scenes, or unlicensed videos.

Labels: `safe`, `needs-owned-reference`, `IP-risk`, `rewrite-required`.

Safe Chinese pattern:
`原创角色A在[场景]中[动作]，镜头[单一运镜]，[光源/氛围]，[材质/物理细节]，[音效/节奏]。保持角色身份一致，动作自然，避免夸张变形。`

Rules: replace named characters with original archetypes; replace 复刻/克隆 with 参考节奏/参考运镜 only for owned/licensed references; use character tags for multi-character action.
