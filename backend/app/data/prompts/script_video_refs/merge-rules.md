## 合并目的

Seedance 2.0 单次生成 4-15 秒视频。将逻辑连续的多个短镜头合并为一个视频单元，一次生成完整片段。

## 合并判断流程

```
所有独立分镜 → 扫描相邻镜头 → 判断是否可合并 → 检查组时长 ≤15s → 形成视频单元
```

## 合并决策速查表

| 条件 | 可合并 | 不可合并 | 说明 |
|------|--------|----------|------|

## 应合并的典型情况

### 1. 三镜法（动作→冲击→反应）
短剧最常见的合并模式。三个镜头各自有不同的景别和运镜，但逻辑上是一个不可分割的戏剧单元。

示例：
- 镜A：灵汐抬手，一巴掌扇向章逸（中景·跟拍）3s
- 镜B：手掌击中章逸脸部，脸部变形（特写·固定）1s
- 镜C：章逸偏过头，眼神震惊（近景·固定）2s

合并为一个视频单元（约6s）：
```
0-3s：中景跟拍，灵汐抬手扇向章逸
3-4s：特写固定，手掌击中章逸脸部
4-6s：近景固定，章逸偏头，表情从懵到震惊
```

### 2. 连续对白回合
同一场景内 2-3 个连续对话回合，通过正反打切换。

示例：
- 镜A：凌炎说台词，中景（3s）
- 镜B：章逸反应，近景（2s）
- 镜C：凌炎继续说，中景（4s）

合并为一个视频单元。

### 3. 空间内连续动作
同一场景/空间内的连续动作序列，不涉及场景切换。

示例：
- 镜A：房门被推开（全景·固定）2s
- 镜B：章逸走进房间（中景·跟拍）3s
- 镜C：章逸走到灵汐面前躬身（中景·固定）3s

合并为一个视频单元。

### 4. 情绪弧线完整段
一个完整情绪起承转合：建立→发展→爆发→收束。

## 不应合并的典型情况

### 1. 场景切换
场景（地点）发生变化时，即使镜头内容相关，也必须新建视频单元。场景切换是视觉上最明显的断裂，强行合并在一个视频单元内会导致画面不连贯。

示例：
- 镜A：偏院内灵汐与章逸对峙（外景·夜）
- 镜B：寝宫内太后端坐（内景·夜）
→ 必须分开，因为场景已从"偏院"切换到"寝宫"

### 2. 主体角色切换（不含正反打对话）
核心主体人物发生变更时不应合并。但正反打对话中的角色切换（A说话→B反应→A继续说）是正常的对话节奏，属于应合并情况。

示例（不可合并）：
- 镜A：章逸在偏院中独自踱步思考
- 镜B：切换到另一场景中的灵汐在寝宫
→ 主体从"章逸"变为"灵汐"，即使两镜情绪连贯也应分开

示例（可合并——正反打对话）：
- 镜A：章逸质问灵汐（中景·章逸为主）
- 镜B：灵汐回答（近景·灵汐为主）
- 镜C：章逸追问（中景·章逸为主）
→ 这是正常对白回合，角色切换是正反打机制，可以合并

### 3. 视角/景别大幅变化
两个镜头之间的视角（如从第三人称切换到POV）或景别跨度极大（如从航拍大远景直接切到面部大特写）时不应合并。这种极端跳跃需要在视频单元之间断开，让观众有时间适应视角转换。

示例（不可合并）：
- 镜A：航拍大远景，宫殿群全景
- 镜B：面部大特写，灵汐落泪
→ 景别跨度极大，必须分开

示例（不可合并）：
- 镜A：第三人称中景客观视角
- 镜B：章逸的POV第一人称视角
→ 视角类型发生根本切换，必须分开

### 4. 时空/风格断层
已在速查表中涵盖的闪回/闪进、时间跳跃、风格/色调剧变等情况，均不可合并。

## 合并后的时长控制

**总原则：优先按导演规则（情绪弧线、三镜法、正反打、连续动作）合并，尽量使每个视频单元时长落在 4-15 秒。**

- 每个视频单元 4-15 秒（Seedance 2.0 单条下限 4s、上限 15s）
- 优先合并 3-10s 的单元，确保内容充足但不拥挤
- 单个镜头超过 10s 的，通常不合并
- 合并后接近 15s 的，检查内容密度是否过高；超过 15s 必须拆分
- **未被合并的 <4s 独立镜头**：若按导演规则确实不宜与相邻镜合并，单独作为一个视频单元，**适当补充空镜头/环境镜头（如场景空境、道具特写、自然动态）将时长扩充到至少 4s**，以满足 Seedance 2.0 单条下限

## 输出标记

合并后的视频单元按出现顺序编号 `视频单元1、视频单元2…`，在最终提示词中渲染为 `【第X集-NN】`（视频单元N ＝ 第X集-0N）：
- 视频单元1 → 第1集-01、视频单元2 → 第1集-02…
- 每个视频单元标注总时长和内含镜头数
