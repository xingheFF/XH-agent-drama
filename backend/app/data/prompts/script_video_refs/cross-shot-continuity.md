# 相邻镜头连贯性检查

## 检查项

### 1. 空间连贯性
- **180度线**：对话场景中，机位不可越过两人连线
- **银幕方向**：人物从左向右走，下一镜头不可变成从右向左
- **空间关系**：上一镜头在门外，下一镜头不可突然在屋内（除非有转场）

### 2. 时间连贯性
- **动作衔接**：上一镜头手伸向门把，下一镜头门应处于打开/正在打开状态
- **道具状态**：上一镜头拿着杯子，下一镜头杯子不可消失（除非有交代）
- **角色位置**：上一镜头站着，下一镜头不可突然坐着

### 3. 视线连贯性
- **眼线匹配**：A 看向画面右侧，下一个 B 的镜头 B 应在画面左侧（相对位置）
- **看物方向**：角色看向某物，下一镜头该物应出现在角色视线方向

### 4. 情绪连贯性
- **情绪过渡**：不跳跃过大（如从"平静"直接跳到"暴怒"需有过渡）
- **节奏递进**：景别由远到近时，情绪应递进；由近到远时，应释放

### 5. 技术连贯性
- **光线一致**：同一场景同一时间，光线方向和色温一致
- **色调一致**：同一场景的调色风格一致
- **服装一致**：同一场景同一角色服装不变

## 连贯性违规处理

| 违规类型 | 处理方式 |
|---|---|
| 方向跳跃 | 加插入镜头或调整运镜方向 |
| 动作不接 | 修改镜头描述，补充过渡动作 |
| 道具消失 | 在镜头描述中补充道具交代 |
| 光线不一致 | 统一光线设定或标注时间变化 |
| 情绪跳跃 | 增加过渡镜头或调整情绪曲线 |
