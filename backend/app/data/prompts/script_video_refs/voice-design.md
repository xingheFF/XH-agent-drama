# 音色设计

## 设计原则

为每个有台词或OS（旁白/内心独白）的角色设计音色，使AI生成或TTS配音时有明确参考。

## 音色维度

每个角色音色需描述以下维度：
1. **性别**：男/女
2. **年龄感**：如"30岁""老年""少年"
3. **核心特质**（2-3个词）：如"沉稳、略带沙哑""清亮、活泼""低沉、威严"

## 常见角色类型音色参考

| 角色类型 | 性别 | 年龄感 | 核心特质 |
|---|---|---|---|
| 霸道总裁 | 男 | 30-35岁 | 低沉、磁性、克制 |
| 甜宠女主 | 女 | 22-28岁 | 清甜、柔软、略带气声 |
| 反派/权臣 | 男 | 40-50岁 | 阴沉、低沉、压迫感 |
| 少年/学生 | 男 | 16-20岁 | 清亮、活力、略带沙哑 |
| 知性女性 | 女 | 30-35岁 | 沉稳、温和、清晰 |
| 老者/师父 | 男/女 | 60岁+ | 苍老、沙哑、缓慢 |
| 旁白/叙述者 | 男 | 35-45岁 | 沉稳、略带沙哑、慢语速 |
| 活泼闺蜜 | 女 | 20-25岁 | 明亮、快速、跳跃 |

## 台词格式规范

台词在视频提示词中的格式：
```
角色名（动作）：台词内容
```

例如：
```
陈明（低头看照片，声音微颤）："这个人...是我？"
```

## OS（内心独白）格式
```
陈明（OS）：（内心独白，不出声）这怎么可能...
```

## 台词长度建议
- 单条台词不超过 15 个字（5秒内可说完）
- 对话镜头台词不超过 2 轮
- 旁白按镜头时长匹配：slow=3字/秒, normal=3.5字/秒, fast=4.5字/秒
