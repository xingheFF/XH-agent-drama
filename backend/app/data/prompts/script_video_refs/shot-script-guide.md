## 一、景别选择指南

| 景别 | 画面范围 | 短剧典型用途 |
|------|----------|-------------|
| 大远景 | 全景地貌 | 场景建立、朝代切换 |
| 全景 | 人物全身+环境 | 人物入场、群戏 |
| 中景 | 腰部以上 | 常规对话、互动 |
| 近景 | 胸部以上 | 情感对话、微表情 |
| 特写 | 面部 | 情绪爆发、震惊反应 |
| 大特写 | 眼睛/手/物品 | 关键道具、眼神戏 |

## 一之二、视角选择指南

视角决定了观众"通过谁的眼睛"看画面，直接影响代入感和信息揭示方式：

| 视角类型 | 描述 | 短剧典型用途 |
|----------|------|-------------|
| 第三人称客观 | 摄影机作为旁观者，不代入任何角色 | 绝大多数常规叙事镜头 |
| POV第一人称 | 通过某个角色的眼睛看，画面即该角色的所见 | 悬疑发现、震惊反应、关键信息揭示 |
| 过肩视角 | 从角色肩后拍摄，画面含该角色部分身体 | 正反打对话、对峙戏 |
| 俯视上帝视角 | 从正上方垂直向下拍摄 | 全景俯瞰、命运转折、群像入场 |
| 窥视视角 | 通过门缝/窗格/幕帘等遮挡物拍摄 | 偷听、暗中观察、悬疑氛围 |
| 仰视视角 | 从低处向上看 | 表现角色的威严/压迫感，或弱小者的仰望 |
| 俯视视角 | 从高处向下看 | 表现角色的卑微/孤独，或审视感 |

**视角选择要点**：
- 一场戏通常以第三人称客观视角为主，在关键时刻切换POV或特殊视角增强戏剧效果
- POV镜头必须明确标注"谁的POV"（如"章逸POV"、"灵汐POV"）
- 正反打对话默认使用过肩视角，若用纯客观视角需特别说明
- 特殊视角（上帝视角、窥视视角）不轻易使用，仅在叙事需要时切换
- 同一视频单元内视角切换不超过2次，保持视觉连贯

## 二、短剧运镜常用手法

### 2.1 基础运镜
- **推 (Push in)**：逐渐靠近主体，增强情绪强度
- **拉 (Pull back)**：逐渐远离，揭示更大空间或表现孤独
- **摇 (Pan)**：水平旋转，跟随人物或展示空间
- **移 (Track/Dolly)**：摄影机平移跟随主体
- **跟 (Follow shot)**：保持与主体相对位置不变
- **升/降 (Crane up/down)**：垂直升降，改变视角

### 2.2 短剧高频运镜
- **急推 (Snap push)**：快速推向人脸 → 震惊/愤怒/反转时刻
- **甩镜头 (Whip pan)**：快速甩向另一主体 → 转场/节奏切换
- **手持微晃 (Handheld slight)**：轻微晃动 → 紧张/慌乱感
- **固定 (Static)**：三脚架固定 → 冷静观察/对峙/威严感
- **低角度 (Low angle)**：仰拍 → 权力感/压迫感
- **高角度 (High angle)**：俯拍 → 弱势/被审视
- **POV (第一人称)**：角色视角 → 代入感

## 三、光影与氛围

### 3.1 古装短剧光影类型
| 光影类型 | 效果 | 场景举例 |
|----------|------|----------|
| 暖黄烛光 | 暧昧、宫廷夜戏 | 寝宫夜戏 |
| 冷蓝月光 | 清冷、阴谋、孤独 | 偏院夜戏 |
| 柔光高调 | 唯美、闪回、仙境 | 回忆/系统画面 |
| 侧光硬光 | 戏剧冲突、对峙 | 对峙戏 |
| 逆光剪影 | 神秘、威严、悬念 | 角色出场 |

### 3.2 现代剧光影类型
| 光影类型 | 效果 | 场景举例 |
|----------|------|----------|
| 办公室荧光灯 | 冷白、压抑、职场感 | 办公室对峙/加班 |
| 街灯暖黄 | 都市夜归、孤独、温情 | 夜晚街头独处 |
| 霓虹灯光 | 赛博感、诱惑、迷离 | 酒吧/夜店/都市夜景 |
| 日光自然光 | 明亮、日常、清新 | 户外约会/日常生活 |
| 台灯柔光 | 私密、温馨、深夜 | 卧室夜话/独处沉思 |
| 手机/电脑屏幕光 | 冷蓝、焦虑、信息过载 | 深夜刷手机/办公 |

### 3.3 奇幻剧光影类型
| 光影类型 | 效果 | 场景举例 |
|----------|------|----------|
| 魔法光效（荧光/粒子光） | 超自然、神秘、能量涌动 | 施法/魔法觉醒 |
| CG融合光（全息/数据流光） | 未来感、虚拟与现实交织 | 系统界面/异空间 |
| 圣光/神光 | 威严、神圣、净化 | 神殿/天界/神降 |
| 幽冥绿光/紫光 | 邪恶、危险、亡灵 | 地下城/暗黑法师/诅咒 |
| 灵体微光 | 朦胧、非现实、灵魂感 | 灵魂出窍/回忆/梦境 |

### 3.4 色调
- 古装宫廷暖色调（金、红、琥珀）→ 权力、富贵
- 冷色调（蓝、青、灰）→ 阴谋、压迫、夜晚
- 现代暖色调（米黄、橙）→ 温馨、日常
- 现代冷色调（灰白、蓝白）→ 职场、疏离
- 奇幻暖色（金、紫红）→ 神圣、魔法
- 奇幻冷色（幽蓝、翠绿）→ 诡异、暗黑

## 四、构图要点

### 4.1 通用构图

- **三分法**：主体放在画面1/3处
- **引导线**：利用建筑线条（廊柱、窗棂）引导视线
- **框架构图**：通过门窗、纱帐框住主体
- **留白**：人物看向的方向留空间
- **景深**：浅景深特写突出人物，深景深全景展示场景

### 4.2 竖屏 9:16 构图要点

短剧以竖屏为主，构图需适配 9:16 画幅：

- **纵向空间利用**：竖屏天然适合展示人物全身站立、上下空间关系（如跪拜、台阶），优先考虑纵向构图
- **人物居中偏下**：将人物主体放在画面中下部，上方留出环境/氛围空间
- **双人构图**：竖屏双人建议上下排列（一立一坐/一站一跪）而非左右并排，若需并排则用近景/特写截取上半身
- **避免横向元素溢出**：广角全景、横向移动的大场景在竖屏中信息损失大，改用中景/近景替代
- **留白方向**：竖屏人物视线上方留白（而非左右），营造纵深感
- **字幕空间**：若后续需加字幕，在画面下方预留 1/6 区域避免遮挡人物面部
- **运镜适配**：推/拉比横摇更自然；跟拍时纵向跟拍优于横向移动

## 五、时长估算标准

| 内容类型 | 参考时长 |
|----------|----------|
| 纯动作（一巴掌、推门） | 2-4s |
| 简单对话（一句话） | 3-6s |
| 带反应的情绪对话 | 4-8s |
| 情绪爆发/反转时刻 | 3-5s |
| 环境建立镜头 | 2-4s |
| 闪回快切（单个） | 1-2s |
| OS内心独白 | 按语速 4字/s 估算 |
