# 视频内容写作规范

## 提示词固定结构（唯一权威格式）

每个视频单元输出一条提示词，结构如下：

```
【{集数}-NN | 总时长Xs】
【参考素材】
@角色A；
@场景B；
@道具C；

【视频风格】
[基础风格：对所有视频单元通用的画面风格描述]
- 时间：[白天/夜晚/黄昏/凌晨]
- 光源：主光源为[...]
- 色温：[描述+大约色温值]
- 动态元素：[如有则描述]
- 氛围：[情绪氛围关键词]
- 微细节：[如有则描述]

【视频内容】
[一段自然连贯的叙事段落，严格基于分镜脚本按时间先后串联所有镜头]

视频约束：
无字幕、无水印、无背景音乐。[定制化约束内容]
```

## 写作规范

### 参考素材段
- 每个角色/场景/道具用 `@名称` 引用
- 用户准备的参考图承担角色外观，**提示词中绝不描写角色外貌**
- 只列出本视频单元涉及的素材

### 视频风格段
- **基础风格**：从 style-guide.md 中选取对应风格的基础描述（全剧统一）
- **时间**：本视频单元发生的时间段
- **光源**：具体到光源类型、方向、颜色
- **色温**：大约色温值（如"暖黄约3200K"）
- **动态元素**：雨、雪、雾、粒子、旗帜飘动等
- **氛围**：2-4个情绪关键词
- **微细节**：衣物材质反光、皮肤质感、金属锈迹等

### 视频内容段
- **自然连贯的叙事段落**，不是分镜列表
- 按时间先后串联所有镜头的动作和事件
- 包含角色的可见动作和台词
- 台词格式：`角色名（动作）："台词"`
- 结尾有明确的终止画面

### 视频约束段
- 固定约束：`无字幕、无水印、无背景音乐`
- 定制化约束（根据镜头内容）：
  - 多人场景：`保持角色外观全程一致`
  - 文字/Logo：`画面中文字保持静态清晰`
  - 连续动作：`动作方向连贯，不跳切`
  - 道具：`[道具名]状态保持一致`
  - 环境动态：`[雨/雪/雾]方向和密度保持一致`

## 常见错误

1. ❌ 在提示词中描写角色外貌 → ✅ 用 @角色名 引用参考图
2. ❌ 分镜列表式写法（"镜头1：... 镜头2：..."）→ ✅ 自然叙事段落
3. ❌ 基础风格含时间限定词（"夜晚暗调"）→ ✅ 时间单列一行
4. ❌ 约束套用固定模板不定制 → ✅ 根据镜头风险定制约束
5. ❌ 台词过长（超过15字）→ ✅ 精简台词
