## 画面风格大类

视频风格首先确定画面风格大类，这是最基础的视觉基调，决定了整体呈现方式。以下为常用大类及其参数。

### 真人写实

追求真实光影、自然质感的影视级画面，类似真人实拍效果。

| 维度 | 参数 |
|------|------|
| 画面质感 | 影视级真实质感，皮肤纹理可见，布料褶皱自然 |
| 光影 | 写实自然光+人工光源模拟，注意光源方向一致 |
| 色彩 | 自然色域，不过度饱和，符合人眼视觉习惯 |
| 景深 | 浅景深突出主体，符合真实镜头的光学特性 |
| Seedance 关键词 | photorealistic, cinematic lighting, natural skin texture, film grain |

### 2D动漫

平面手绘/数字绘画风格，线条清晰、色彩明快。

| 维度 | 参数 |
|------|------|
| 画面质感 | 平面插画质感，线条描边清晰，色块填充均匀 |
| 光影 | 简化光影，多用平涂/渐变阴影，少用真实物理光照 |
| 色彩 | 高饱和、明快色调，色彩层次由色块和渐变表现 |
| 景深 | 浅景深或平面对比，背景可做模糊简化 |
| Seedance 关键词 | 2D anime style, cel shading, clean lineart, flat color, vibrant |

### 赛博朋克

未来科技感与低端生活的碰撞，霓虹灯+暗黑都市美学。

| 维度 | 参数 |
|------|------|
| 画面质感 | 金属反光、玻璃透射、全息投影的科技质感 |
| 光影 | 霓虹灯光为主，多色光源（粉紫+青蓝），强烈明暗对比 |
| 色彩 | 暗部偏冷蓝/紫，亮部霓虹粉/橙，高对比低饱和暗部 |
| 景深 | 浅景深突出主体，背景霓虹虚化为光斑 |
| Seedance 关键词 | cyberpunk aesthetic, neon lighting, rain-slicked streets, holographic glow, high contrast |

### 水墨国风

中国传统水墨画审美，写意留白、墨色层次。

| 维度 | 参数 |
|------|------|
| 画面质感 | 宣纸纹理/绢本质感，墨色浓淡干湿变化 |
| 光影 | 淡化物理光影，以墨色深浅代替明暗，留白代替高光 |
| 色彩 | 黑白灰为主，朱砂/石青/藤黄点缀，低饱和高雅 |
| 景深 | 写意留白，以虚写实，近浓远淡 |
| Seedance 关键词 | Chinese ink wash painting, sumi-e style, rice paper texture, minimalist, ethereal |

### 复古胶片

模拟胶片摄影/老电影质感，颗粒感+偏色。

| 维度 | 参数 |
|------|------|
| 画面质感 | 胶片颗粒感，轻微刮痕/漏光，柔焦晕染 |
| 光影 | 柔光为主，高光微微溢出，暗部偏蓝/绿 |
| 色彩 | 偏暖黄或偏冷青，低饱和褪色感，肤色偏暖 |
| 景深 | 中浅景深，边缘柔焦 |
| Seedance 关键词 | vintage film look, film grain, soft focus, warm color cast, analog aesthetic |

## 画面质感维度

画面质感描述图像给人的触觉联想和细节密度，是画面风格的核心维度之一：

| 质感类型 | 描述 | 适用场景 |
|----------|------|----------|
| 影视级写实 | 高清锐利，皮肤/布料/金属纹理真实 | 真人写实类 |
| 柔和朦胧 | 轻微柔焦，画面像蒙了一层薄纱 | 回忆/梦境/浪漫 |
| 粗粝纪实 | 高颗粒感、手持晃动的纪录片质感 | 悬疑/战争/纪实 |
| 光滑插画 | 边缘清晰、色块平涂的矢量感 | 2D动漫/扁平风 |
| 金属科技 | 反光高光强烈、冷硬质感 | 赛博朋克/科幻 |
| 水墨晕染 | 墨色浓淡渗化、笔触飞白 | 水墨国风 |
| 胶片颗粒 | 随机颗粒分布、轻微漏光偏色 | 复古/怀旧 |

## 光影效果设计

光影决定了画面的立体感和戏剧张力。在视频风格描述中需明确：

- **光源类型**：自然光（日光/月光）/ 人工光（烛光/霓虹/荧光灯）/ 特效光（魔法光/全息光）
- **光质**：硬光（明暗分明、对比强）/ 柔光（过渡柔和、阴影浅）
- **光位**：正面光 / 侧光 / 逆光 / 顶光 / 底光
- **光比**：高光与暗部的亮度比例，大光比=强烈戏剧感，小光比=柔和扁平

## 色彩影调设计

色彩决定了画面的情绪基调：

| 影调类型 | 特征 | 情绪联想 |
|----------|------|----------|
| 暖金调 | 金、琥珀、红棕为主 | 权力、富贵、温暖、怀旧 |
| 冷蓝调 | 蓝、青、灰为主 | 孤独、阴谋、冷静、夜晚 |
| 粉柔调 | 粉、奶白、浅紫 | 甜蜜、浪漫、少女、梦幻 |
| 暗绿调 | 墨绿、褐、暗灰 | 诡异、压抑、自然、悬疑 |
| 高饱和 | 色彩浓郁、冲击力强 | 热血、奇幻、动漫感 |
| 低饱和 | 色彩收敛、接近灰阶 | 严肃、写实、压抑、高级感 |
| 霓虹调 | 粉紫+青蓝+橙红高对比 | 赛博、夜生活、未来感 |

## 情绪氛围设计

情绪氛围是画面给人的心理感受，需在视频风格中明确：

| 氛围 | 画面表达方式 |
|------|-------------|
| 压抑紧张 | 低光、大光比、冷色调、手持微晃 |
| 温馨甜蜜 | 柔光、暖色调、浅景深、稳定运镜 |
| 神秘诡异 | 逆光剪影、局部照明、暗绿/紫色调 |
| 壮阔史诗 | 大景深、暖金色调、升降运镜、粒子光 |
| 清冷孤寂 | 冷蓝月光、大面积留白/负空间、深景深 |
| 热烈奔放 | 高饱和、强光照、快速运镜、暖橙色调 |

## 风格组合示例

以下为剧情类型与画面风格大类的组合示例，实际设计时以画面风格维度为主，剧情类型决定具体参数选择。

### 古装宫斗·真人写实

| 维度 | 参数 |
|------|------|
| 画面风格大类 | 真人写实 |
| 画面质感 | 影视级写实，古装绸缎/木质纹理，空气中有轻微尘埃 |
| 光影效果 | 暖黄烛光+冷蓝月光交替，侧光硬光增强对峙感 |
| 色彩影调 | 暖金+暗红为主，暗部偏冷蓝，低饱和深沉 |
| 情绪氛围 | 权谋压抑，烛光摇曳增添不安 |

### 甜宠轻喜·真人写实

| 维度 | 参数 |
|------|------|
| 画面风格大类 | 真人写实 |
| 画面质感 | 柔和朦胧偏写实，皮肤光洁带柔光感 |
| 光影效果 | 柔光高调，自然光+暖色补光，低光比 |
| 色彩影调 | 粉柔调，暖粉+奶白+浅金，明亮高调 |
| 情绪氛围 | 轻松甜蜜，画面温暖轻盈 |

### 悬疑暗黑·真人写实

| 维度 | 参数 |
|------|------|
| 画面风格大类 | 真人写实 |
| 画面质感 | 粗粝偏写实，暗部细节丰富，可加轻微颗粒感 |
| 光影效果 | 低光硬光，局部照明，明暗对比强烈 |
| 色彩影调 | 冷蓝+深灰+暗绿，低饱和 |
| 情绪氛围 | 紧张压迫，快慢节奏交替制造不安 |

### 都市现代·真人写实

| 维度 | 参数 |
|------|------|
| 画面风格大类 | 真人写实 |
| 画面质感 | 影视级写实，现代服装/建筑纹理真实 |
| 光影效果 | 自然光+室内人工光交替，写实混合光源 |
| 色彩影调 | 灰白+蓝+暖黄，写实色域 |
| 情绪氛围 | 根据具体场次切换，职场冷感/情感温馨 |

### 奇幻玄幻·真人写实

| 维度 | 参数 |
|------|------|
| 画面风格大类 | 真人写实（可混合 CG 元素） |
| 画面质感 | 写实底+魔法/粒子特效层，CG 元素与实拍融合 |
| 光影效果 | 魔法光效（荧光/粒子光）+逆光+圣光，多源特效光 |
| 色彩影调 | 深紫+金+幽蓝，高饱和神秘 |
| 情绪氛围 | 超自然壮阔，史诗感与神秘感交替 |

## 向用户确认

自动设计的画面风格以一句话概括，在输出前展示给用户确认：

> "根据剧本内容，我为本集设计的画面风格是：【风格大类 + 画面质感 + 光影/色调 + 情绪氛围】。是否合适？如需调整请告诉我。"

例如：
> "根据剧本内容，我为本集设计的画面风格是：真人写实古装质感，暖金暗红色调配合冷蓝月光，侧光硬光增强对峙感，情绪压抑紧张。是否合适？"
