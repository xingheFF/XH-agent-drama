# 画面风格设计

## 基础风格设计原则

基础画面风格描述必须是对**全剧所有视频单元都适用**的通用描述，不得包含：
- 受时间段制约的描述（"白天""夜晚"——这些放在每个视频单元的时间段中）
- 受室内外制约的描述（"室内暖光""室外自然光"——这些放在每个视频单元的光源中）
- 受特定天气制约的描述（"雨天""雪天"——除非全剧都是同一天气）

## 各风格基础描述

### 真人写实
```
写实电影感，自然皮肤质感，35mm镜头视角，浅景深，柔和色调过渡，PBR物理材质渲染
```

### 古装真人写实
```
古典东方美学，丝绸与棉麻质感，低饱和暖调，柔光散射，古代建筑材质写实，烛火/自然光氛围
```

### 现代都市写实
```
现代都市写实，玻璃与金属反射，混合光源（日光+荧光灯+霓虹），轻度去饱和，当代服装质感
```

### 悬疑暗黑写实
```
暗调高对比，冷色低饱和，深阴影保留细节，硬光切割画面，烟尘氛围颗粒感
```

### 甜宠轻喜写实
```
明亮高调，暖色调偏粉，柔光填充，浅景深背景虚化，干净通透肤色，光晕溢出
```

### 奇幻玄幻写实
```
超写实奇幻，体积光与粒子特效，高饱和魔幻色调，金属与水晶材质，光效层次丰富
```

### 2D动漫
```
2D赛璐珞动画风格，平涂色块+赛璐珞阴影，清晰线稿，有限运动模糊，鲜艳调色板
```

### 赛博朋克
```
霓虹浸染赛博朋克，品红与青色光源，雨面反射，全息投影光效，高对比阴影，金属与LED材质
```

### 水墨国风
```
中国水墨画风，动态笔触，留白构图，墨色浓淡层次，宣纸质感，写意山水
```

### 复古胶片
```
复古胶片质感，柯达Portra 400暖色，轻微光晕，柔对比，褪色怀旧色调，35mm胶片颗粒
```
