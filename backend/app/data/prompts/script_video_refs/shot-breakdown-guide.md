## 一、导演思维分镜原则

### 1.1 以情绪和叙事驱动镜头拆分
不要机械地按剧本△动作块拆分。导演思维的核心是：
- **情绪弧线**：一个情绪爆发、转折、收束 = 一个镜头序列
- **信息揭示**：每次镜头切换应揭示新信息或推进情绪
- **节奏控制**：紧张戏多用短镜头快切，抒情戏用长镜头

### 1.2 镜头拆分粒度
- 短剧节奏快，单个镜头一般 2-8 秒
- 一个"动作单元" = 1 个镜头
- 三镜法：动作→冲击→反应，每个环节拆为独立镜头

### 1.3 台词与动作不分离原则

**核心规则**：角色的动作/状态描述与紧随其后的台词，若在剧本中属于同一叙事段落，必须保持在同一个镜头内，不可拆分为两个独立镜头。

**判断标准**：
- 剧本中"动作/状态描述 + 角色台词"紧密相邻（同一段落/同一行），且动作/状态是该角色说出台词的即时伴随状态 → 不可拆分
- 一句台词独自一行，前后无同一角色的动作描述 → 可独立成镜

**示例**：

❌ 错误拆分：
```
镜A：灵汐躺在床上双眼紧闭，眉头微蹙，喃喃自语。
镜B：灵汐："老板，方案我还能改，今晚给到……"
```

✅ 正确：
```
镜A：灵汐躺在床上双眼紧闭，眉头微蹙，喃喃自语。
     灵汐："老板，方案我还能改，今晚给到……"
```

> 喃喃自语的动作状态与台词内容是一个不可分割的表演整体，拆分会导致表演断裂——镜头A中角色在说什么？镜头B中角色在做什么？两者互为因果，必须同一镜头。

### 1.4 常见镜头序列模式
| 模式 | 描述 | 适用场景 |
|------|------|----------|
| 建立→进入 | 全景建立空间 → 中景进入人物 | 场景开头 |
| 正反打 | A说话→B反应→A继续说 | 对话 |
| 三镜法 | 动作→冲击→反应 | 冲突/打脸/反转 |
| 累积式 | 多个短镜头快速切换 | 紧张/震惊/闪回 |
| 揭示式 | 遮挡→移开→揭示 | 悬疑/反转/震惊 |
