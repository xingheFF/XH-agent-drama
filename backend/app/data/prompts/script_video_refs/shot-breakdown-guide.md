# 分镜拆解指南

## 拆解原则

### 基本规则
1. **以场为单位拆解**：每个场景内按叙事节奏拆为独立镜头
2. **一镜一动作**：每个镜头聚焦一个核心动作或情绪表达
3. **时长匹配**：单镜头 2-6 秒，片头/片尾空镜可至 8 秒
4. **景别节奏**：避免连续相同景别，推荐"大→中→特"或"特→中→大"的呼吸节奏
5. **转场设计**：情绪转折处用 match_cut 或 dissolve，日常衔接用 cut

### 拆解步骤
1. 通读剧本，标注每个场景的起止点
2. 在场景内按"动作单元"切分镜头
3. 为每个镜头标注：景别、运镜、时长、叙事功能
4. 检查相邻镜头的连贯性（方向、视线、空间关系）
5. 验证总时长是否在目标 ±10% 范围内

### 景别选择指导
| 叙事功能 | 推荐景别 | 说明 |
|---|---|---|
| 交代环境 | 大全景/全景 | 场景建立，空间关系 |
| 人物互动 | 中景/中近景 | 关系与距离 |
| 情绪表达 | 近景/特写 | 微表情与内心 |
| 关键细节 | 特写/大特写 | 道具、动作细节 |
| 压迫/孤立 | 远景+负空间 | 环境压倒人物 |

### 运镜选择指导
| 情绪/功能 | 推荐运镜 | 说明 |
|---|---|---|
| 发现/领悟 | 推镜 | 聚焦，靠近 |
| 揭示/放弃 | 拉镜 | 远离，揭示空间 |
| 跟随/追踪 | 跟拍 | 伴随主体运动 |
| 环绕/展示 | 环绕 | 全方位展示 |
| 稳定/克制 | 固定 | 观察者视角 |
| 紧张/纪实 | 手持 | 不稳定感 |
| 规模/到达 | 航拍/升降 | 空间纵深 |
