# 画风预设库

## 写实
```
photorealistic, natural skin texture, realistic lighting, 35mm photography, shallow depth of field
```

## 电影质感现实主义
```
cinematic realism, anamorphic lens flare, film grain, teal and orange grade, 2.39:1 aspect feel
```

## 王家卫风格
```
saturated colors, step-printing motion blur, tight close-ups, warm tungsten light, melancholic atmosphere, neon reflections
```

## 纪录片/手持写实
```
documentary handheld, natural available light, slightly desaturated, 16mm grain, candid framing
```

## 古风写实
```
classical Chinese aesthetic, muted earth tones, natural silk textures, ink-wash atmosphere, soft diffused light
```

## 古风唯美
```
romantic ancient Chinese style, flowing silk garments, cherry blossom particles, golden hour backlight, ethereal glow
```

## 古朴水墨
```
traditional Chinese ink painting, monochrome with selective color, rice paper texture, brush stroke aesthetics, vast empty space
```

## 现代都市写实
```
contemporary urban realism, glass and steel reflections, fluorescent mixed with warm practical light, smartphone-era color grade
```

## 赛博朋克
```
neon-soaked cyberpunk, magenta and cyan lighting, rain-slicked streets, holographic reflections, high contrast shadows
```

## 轻奢/商业时尚

luxury commercial, clean studio lighting, marble and gold textures, shallow depth of field, editorial color grade

vintage film look, Kodak Portra 400 warmth, slight halation, soft contrast, nostalgic faded colors, 4:3 aspect feel

film noir, hard chiaroscuro lighting, venetian blind shadows, black and white high contrast, cigarette smoke, low-angle drama

Studio Ghibli style, hand-painted backgrounds, soft watercolor textures, lush nature, warm light, gentle atmosphere
