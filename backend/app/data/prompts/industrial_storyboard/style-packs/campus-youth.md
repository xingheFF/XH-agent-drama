# G08 校园青春

> 校园青春题材。自然日光+逆光金边、欢快跳切+跟跑，营造活力懵懂遗憾的氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 自然日光，多向柔光 |
| 色温 | 5500-6500K 日光白 |
| 明暗氛围 | 高调明亮，无明显阴影 |
| 动态光影效果 | 树影婆娑、逆光金边、操场阳光斑驳 |

**辅助光源**：教室日光灯、走廊窗户透光

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 欢快跳切 | 多个0.5-1秒短镜头快速切换，模拟青春节奏 |
| 跟跑前侧 | 与人物同步跑步，保持1.5米距离，速度1.5-2米/秒 |
| 缓推近景 | 从1.5米中近景缓推至0.5米近景，速度0.4米/秒 |
| 逆光特写 | 逆光下人物发丝金边特写，固定机位 |

**禁用运镜方式**：手持晃动（破坏明快感）、低调暗光（破坏青春感）、长焦压缩（过于疏离）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 活力登场 | 欢快跳切、跟跑前侧、操场阳光斑驳 |
| 承 | 懵懂心动 | 缓推近景、逆光特写、双人对称构图 |
| 转 | 遗憾别离 | 缓拉至全景、人物背影远去、配乐渐弱 |

---

## 4. 色彩特征

| 要素 | 描述 |
|---|---|
| 主色调 | 高饱和暖黄+蓝天青草 |
| 点缀色 | 校服蓝白、操场红跑道、教室米白墙 |
| 饱和度 | 中高饱和（60-75%） |
| 对比度 | 低对比（明暗差1-2档），高调为主 |

---

## 5. 构图特征

| 要素 | 描述 |
|---|---|
| 构图方式 | 横构图+多人错落 |
| 留白习惯 | 上方天空留白30%，人物居中偏下 |
| 前景遮挡 | 树叶、飘落花瓣、书包肩带做前景 |
| 纵深 | 浅景深虚化背景，主体凸显 |

---

## 6. 转场偏好

| 转场方式 | 使用频率 | 适用场景 |
|---|---|---|
| 切 | 高频（80%） | 节奏轻快 |
| 叠化 | 中频（10%） | 时间流逝、毕业回忆 |
| 闪回 | 低频（5%） | 关键回忆 |
| 白闪 | 低频（5%） | 心跳瞬间 |

## 8. 典型镜头模板

### 模板 G08-T1 — 活力登场

```
镜头ID：场号-01
景别：中景
运镜：镜头在主角侧前方1.5米处，与主角保持同步匀速前行（速度约1.5米/秒），从中景维持到中景，时长5秒
场景地点：S编号 校园林荫道外，水泥路面，日光照明
环境光影：自然日光自上方斜射色温6500K冷白，高调明亮，树叶婆娑在地面投射斑驳光斑
出场人物：C001 主角A（林荫道左侧步行，背书包）；C002 主角B（林荫道右侧步行，背书包）
人物动作：C001 双手插校服裤袋步行，书包单肩背带晃动；C002 右手抱课本步行，左手食指无意识抠课本边角
面部表情：C001 眉眼舒展，嘴角微扬（情绪：青春活力）；C002 眉头微蹙，嘴角轻抿（情绪：心事重重）
情绪基调：活力登场
台词/音效：无台词 / 音效：远处学生喧闹、自行车铃声清脆、配乐吉他主旋律渐起
转场方式：切
```

### 模板 G08-T2 — 懵懂心动

```
镜头ID：场号-05
景别：近景 → 特写
运镜：镜头从双人正前方1.5米处中近景，沿水平方向缓推至0.3米处双人特写，速度0.3米/秒，时长4秒
场景地点：S编号 教室窗边内，水磨石地面，窗光照明
环境光影：窗光自左方柔射色温5500K冷白，高调明亮，逆光金边勾勒双人发丝轮廓
出场人物：C001 主角A（窗边课桌左侧正坐）；C002 主角B（窗边课桌右侧正坐，面朝C001）
人物动作：C001 右手握铅笔假装做题，余光偷瞄C002侧脸；C002 偏头看向窗外，左手食指在课本上无意识画圈
面部表情：C001 眉眼柔和，嘴角微扬，眼神偷瞄（情绪：懵懂心动）；C002 眉眼舒展，嘴角轻抿，眼神迷茫（情绪：心事懵懂）
情绪基调：懵懂心动
台词/音效：无台词 / 音效：远处上课铃声、笔尖沙沙声、配乐口琴主旋律渐起
转场方式：白闪
```

### 模板 G08-T3 — 遗憾别离

```
镜头ID：场号-12
景别：中景 → 全景
运镜：镜头从主角正前方1.5米处中近景，沿水平方向缓慢后拉至5米处大远景，速度0.5米/秒，时长8秒
场景地点：S编号 校园门口外，水泥路面，日落照明
环境光影：日落金光自西方平射色温2500K暖橙，中调偏暖，逆光金边勾勒主角发丝轮廓，地面拉长影子
出场人物：C001 主角A（校门口正中背对镜头站立，右手提行李箱）；C002 主角B（校门口内侧5米处，面朝C001）
人物动作：C001 右手提行李箱缓步走向校门外的出租车，每一步行李箱轮子在地面上滚动出声；C002 站立不动，左手攥紧校服下摆
面部表情：C001 不可见（背影）；C002 眉头深锁，嘴角下压，眼眶微红（情绪：遗憾压抑）
情绪基调：遗憾别离
台词/音效：无台词 / 音效：行李箱轮子滚动声、远处出租车鸣笛、配乐吉他主旋律渐弱收尾
转场方式：叠化
```
