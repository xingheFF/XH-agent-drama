# G12 电竞热血

> 电竞热血题材。屏幕蓝光+RGB霓虹、第一人称+快速切，营造热血逆袭登顶的氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 屏幕蓝光+RGB霓虹灯条，多向硬光 |
| 色温 | 主体冷蓝7000K+霓虹紫绿2500K |
| 明暗氛围 | 中调偏暗，屏幕光高调 |
| 动态光影效果 | 屏幕闪烁、RGB灯条渐变、键盘背光跳动 |

**辅助光源**：耳机LED光、鼠标光、机箱内RGB光

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 第一人称主观 | 主观视角，模拟玩家视角 |
| 快速切 | 多个0.3-0.8秒短镜头快速切换，模拟游戏节奏 |
| 急推特写 | 关键操作瞬间急推手指/眼神特写，速度>2米/秒 |
| 环绕屏幕 | 镜头以屏幕为圆心环绕，展示电竞氛围 |

**禁用运镜方式**：缓慢匀速横移（破坏热血感）、固定长镜（节奏过慢）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 热血开战 | 第一人称主观、屏幕快速切、配乐鼓点密集 |
| 承 | 逆袭关键时刻 | 急推手指/眼神特写、屏幕关键操作 |
| 转 | 登顶夺冠 | 急推特写、白闪、观众席爆发 |
| 合 | 沉稳离场 | 缓拉至全景、主角背影长焦压缩 |

---

## 4. 色彩特征

| 要素 | 描述 |
|---|---|
| 主色调 | 霓虹紫蓝绿+高对比 |
| 点缀色 | 屏幕白光、键盘RGB、机箱内RGB |
| 饱和度 | 极高饱和霓虹部分（90%），主体中饱和（50%） |
| 对比度 | 极高对比（明暗差5档） |

---

## 5. 构图特征

| 要素 | 描述 |
|---|---|
| 构图方式 | 屏幕框架+人物半身 |
| 留白习惯 | 屏幕占画面40%，人物占30% |
| 前景遮挡 | 耳机、鼠标线、键盘做前景 |
| 纵深 | 浅景深虚化背景，屏幕和人物凸显 |

---

## 6. 转场偏好

| 转场方式 | 使用频率 | 适用场景 |
|---|---|---|
| 切 | 极高频（90%） | 节奏极快 |
| 白闪 | 中频（5%） | 关键操作爆发 |
| 闪回 | 低频（3%） | 战术回忆 |
| 黑场 | 低频（2%） | 段落留白 |

---

## 7. 音效偏好

| 类型 | 内容 |
|---|---|
| 环境音 | 键盘敲击密集、鼠标点击、观众席呐喊、解说员激情 |
| 动效音 | 游戏音效（枪声/技能/爆炸）、耳机低频嗡鸣 |
| 配乐风格 | 电子鼓点+低频贝斯+留白 |

## 8. 典型镜头模板

### 模板 G12-T1 — 热血开战

```
镜头ID：场号-01
景别：中景
运镜：镜头机位固定于选手侧前方1.5米处，从中景维持到中景，时长4秒，仅做0.1米幅度微调
场景地点：S编号 电竞赛场内，防滑垫地面，屏幕蓝光+RGB霓虹灯条照明
环境光影：屏幕蓝光自前方平射色温7000K冷蓝，RGB霓虹灯条自四周散光色温混合紫绿，中调偏暗，屏幕光高调，键盘背光跳动
出场人物：C001 选手（电竞椅正坐，面朝屏幕，双手置于键盘鼠标）
人物动作：C001 双手在键盘鼠标上快速操作，每秒点击5-8次；双眼紧盯屏幕，瞳孔随屏幕内容快速转动
面部表情：C001 眉头紧锁，嘴角微抿，下颌发力，眼神专注锐利（情绪：专注开战）
情绪基调：热血开战
台词/音效：无台词 / 音效：键盘敲击密集、鼠标点击、远处解说员激情呐喊、配乐电子鼓点渐起
转场方式：切
```

```

情绪基调：登顶夺冠
台词/音效：C001："我们赢了！"（语气爆发狂喜） / 音效：观众席呐喊声炸裂、解说员激情嘶吼、配乐电子鼓点重音炸响
```
