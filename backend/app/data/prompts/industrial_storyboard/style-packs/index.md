## 二、风格包参数说明

每个风格包包含以下 9 个参数模块：

| 模块 | 内容 |
|---|---|
| 1. 光影基调 | 主光源方向+色温+明暗氛围+动态光影效果 |
| 2. 运镜偏好 | 常用运动轨迹+速度倾向+禁用运镜方式 |
| 3. 情绪节奏 | 三段式情绪曲线+各阶段典型镜头 |
| 4. 色彩特征 | 主色调+点缀色+饱和度+对比度 |
| 5. 构图特征 | 对称/不对称+留白习惯+前景遮挡程度 |
| 6. 转场偏好 | 切/叠化/闪回/运镜转场的使用倾向 |
| 7. 音效偏好 | 环境音+动效音+配乐风格的默认设置 |
| 8. 典型镜头模板 | 3-5 个该题材高频使用的镜头模板 |
| 9. 案例片段 | 一个完整段落的分镜应用案例 |

## 三、调用方式

### 方式 1 — 按题材名调用

```
[分镜生成] 题材：古装权谋
<剧本片段>
```

AI 自动加载 `style-packs/ancient-political.md` 中的全部参数。

### 方式 2 — 按编号调用

```
[分镜生成] 题材：G01
<剧本片段>
```

### 方式 3 — 自定义题材（无内置风格包）

```
[分镜生成] 题材：自定义
光影基调：<描述>
运镜偏好：<描述>
情绪节奏：<描述>
色彩特征：<描述>
构图特征：<描述>
<剧本片段>
```

### 方式 4 — 在内置题材上临时覆盖参数

```
[分镜生成] 题材：古装权谋，光影覆盖为冷蓝调
<剧本片段>
```

## 五、跨题材混搭

允许用户混合两个风格包的部分参数，例如：

```
[分镜生成] 题材：古装权谋 + 悬疑罪案
说明：使用古装权谋的服饰和场景，但光影和运镜采用悬疑罪案风格
<剧本片段>
```

AI 加载两个风格包后，按用户说明进行参数合并。

## 六、风格包维护

- 新增风格包：在 `style-packs/` 目录下创建新 `.md` 文件，并在本索引表中添加条目。
- 修改风格包：直接编辑对应 `.md` 文件，无需更新索引（参数详情在文件内）。
- 删除风格包：删除对应 `.md` 文件，并从本索引表中移除条目。
