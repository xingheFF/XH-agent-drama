# G10 战争军旅

> 战争军旅题材。硝烟灰黄+爆炸红闪、手持冲击+快速穿梭，营造壮烈兄弟牺牲的氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 爆炸火光+烟雾散光，多向硬光 |
| 色温 | 主体灰黄4500K+爆炸红橙2000K |
| 明暗氛围 | 极高反差，爆炸瞬间过曝 |
| 动态光影效果 | 爆炸红光闪烁、烟雾中光柱可见、弹道轨迹光痕 |

**辅助光源**：照明弹、燃烧桶火光、远处炮火闪光

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 手持冲击 | 跟随士兵奔跑，幅度0.2-0.5米，频率2-4Hz |
| 快速穿梭 | 镜头在多个掩体间快速移动，速度1.5-2米/秒 |
| 低角度仰拍 | 机位贴近地面30度仰拍，塑造英雄感 |
| 升格特写 | 升格4倍速下，子弹/爆炸/士兵倒地特写 |

**禁用运镜方式**：缓慢匀速横移（破坏紧迫感）、稳定固定长镜（破坏冲击感）

## 8. 典型镜头模板

### 模板 G10-T1 — 壮烈出征

```
镜头ID：场号-01
景别：全景
运镜：镜头机位固定于战壕斜侧方3米处，低角度仰拍30度，从全景维持到全景，时长6秒，仅做0.1米幅度微调
场景地点：S编号 战壕边缘外，泥土地面，晨光+照明弹照明
环境光影：晨光自东方平射色温4500K冷白，照明弹自上方散光色温3000K暖黄，极高反差，烟雾中光柱可见
出场人物：C001 连长（战壕边缘正立，背对镜头）；多名士兵（战壕内趴伏）
人物动作：C001 右手挥向远方下令冲锋；多名士兵起身翻过战壕边缘
面部表情：C001 不可见（背影），下颌线条紧绷
情绪基调：壮烈出征
台词/音效：C001："冲！"（语气凌厉爆发） / 音效：士兵呐喊声、步枪上膛咔嗒、远处炮声轰鸣、配乐大编制交响乐渐起
转场方式：白闪
```

### 模板 G10-T2 — 兄弟并肩

```
镜头ID：场号-05
景别：中景
运镜：镜头以手持方式在双人侧前方1-1.5米范围内做不规则微动（幅度0.2-0.3米，频率3Hz），从中景维持到中景，时长5秒
场景地点：S编号 战场废墟内，碎石地面，爆炸火光+烟雾散光照明
环境光影：爆炸火光自左方散光色温2000K暖红橙，烟雾中光柱可见色温4500K冷白，极高反差，爆炸瞬间过曝
出场人物：C002 老兵（废墟墙后正蹲，左手捂右腿伤）；C003 新兵（C002右侧正蹲，右手持步枪警戒）
人物动作：C002 左手捂右腿伤，右手颤抖地递给C003一枚怀表；C003 左手接过怀表攥紧，右手持步枪警戒
面部表情：C002 眉头深锁，嘴角渗血，眼神涣散（情绪：濒死托付）；C003 眉头紧锁，眼眶微红，下颌发力（情绪：悲痛承接）
情绪基调：兄弟托付
台词/音效：C002："……给我妈写信。"（语气微弱颤抖） / 音效：远处步枪连发、弹壳落地叮当、配乐军鼓渐强
转场方式：切
```

```

台词/音效：无台词 / 音效：远处炮声渐稀、烟雾中余烬噼啪、配乐大提琴独奏渐弱收尾
```
