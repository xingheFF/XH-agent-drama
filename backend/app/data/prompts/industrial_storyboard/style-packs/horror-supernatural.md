# G11 惊悚灵异

> 惊悚灵异题材。烛火幽绿+月光冷白、慢推+突然甩镜，营造压抑惊悚绝望的氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 烛火+月光+闪电，单一方向硬光 |
| 色温 | 主体幽绿冷白6500K+烛火暖橙2500K |
| 明暗氛围 | 极低调光，暗部占画面80%以上 |
| 动态光影效果 | 烛火摇曳、闪电瞬间过曝、灵体幽光扩散 |

**辅助光源**：手机屏幕蓝光、烟头红点、远处霓虹

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 慢推低调 | 从3米中景缓推至0.5米近景，速度0.2米/秒 |
| 突然甩镜 | 主体A急速甩向主体B，速度>3米/秒 |
| 主观窥视 | 主观视角越过门框/窗框/百叶窗缓慢推进 |
| 固定长镜 | 机位完全固定，时长5-10秒，营造窒息感 |

**禁用运镜方式**：明调高光（破坏惊悚感）、欢快跳切（破坏压抑感）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 压抑铺垫 | 慢推空镜、低调光下细节特写、环境音密集 |
| 承 | 惊悚现身 | 急推特写、突然甩镜、灵体幽光扩散 |
| 转 | 绝望逃亡 | 手持跟拍奔跑、低角度仰拍追击者 |
| 合 | 余韵未消 | 缓拉至全景、人物定格、配乐未止 |

## 8. 典型镜头模板

### 模板 G11-T1 — 压抑铺垫

```
镜头ID：场号-01
景别：全景
运镜：镜头机位固定于房间门口1.5米处，水平方向自左向右匀速旋转30度，速度0.2米/秒，从全景过渡到中景
场景地点：S编号 废弃老宅客厅内，灰尘满布的木地板地面，月光+烛火照明
环境光影：月光自破损窗户斜射色温6500K冷白，烛火自茶几上方平射色温2500K暖橙，极低调光，烛火摇曳产生动态阴影
出场人物：无
人物动作：无
面部表情：无
情绪基调：压抑铺垫
台词/音效：无台词 / 音效：滴水声节奏稳定、电流嗡鸣低频、远处犬吠一声
转场方式：黑场
```

### 模板 G11-T2 — 惊悚现身

```
镜头ID：场号-06
景别：中景 → 特写
运镜：镜头从主体A正前方2米处中景，沿水平方向急速甩向主体B侧方0.5米处特写，速度>3米/秒，时长0.5秒
场景地点：S编号 废弃老宅走廊内，木地板地面，烛火+月光照明
环境光影：烛火自主角手中平射色温2500K暖橙，月光自走廊尽头窗户平射色温6500K冷白，极低调光，灵体幽绿光自C002周身扩散
出场人物：C001 主角（走廊中央正立，右手持烛）；C002 灵体（走廊尽头背对镜头站立，周身幽绿光晕）
人物动作：C001 右手骤然攥紧烛台，脚步停滞；C002 缓缓转身，黑发垂落遮蔽大半张脸
面部表情：C001 眉头紧锁，瞳孔骤缩，下颌发力，嘴唇颤抖（情绪：惊悚战栗）；C002 露出的半张脸嘴角缓缓上扬至耳根（情绪：诡异微笑）
情绪基调：惊悚现身
台词/音效：无台词 / 音效：烛火骤然熄灭轻响、心跳声急促、配乐弦乐不和谐音炸响
转场方式：黑场
```

### 模板 G11-T3 — 绝望逃亡

```
镜头ID：场号-09
景别：中景
运镜：镜头以手持方式在主角侧前方1-1.5米范围内做不规则微动（幅度0.2-0.3米，频率4Hz），与主角保持同步奔跑（速度2米/秒），从中景维持到中景，时长6秒
场景地点：S编号 废弃老宅楼梯内外，破旧木板地面，烛火+月光照明
环境光影：烛火已灭，月光自楼梯上方天窗平射色温6500K冷白，极低调光，楼梯木板上月光投射矩形光斑
出场人物：C001 主角（楼梯上奔跑下楼，右手扶楼梯栏杆）；C002 灵体（楼梯顶端缓慢飘下，周身幽绿光晕）

情绪基调：绝望逃亡
台词/音效：无台词 / 音效：脚步急促踏木板声、木板吱呀作响、灵体幽绿光晕嗡鸣、配乐低频持续音渐强

```
