# G07 仙侠玄幻

> 仙侠玄幻题材。仙气冷光+法术彩光、威亚俯仰+升格，营造出尘激战归隐的氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 仙气漫射光+法术点光源 |
| 色温 | 主体冷白7000K+法术多彩光 |
| 明暗氛围 | 高调明亮+局部彩光斑 |
| 动态光影效果 | 仙气流动、法术光晕扩散、剑气光痕 |

**辅助光源**：法宝发光、灵兽瞳光、远处仙山金光

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 威亚俯仰 | 镜头跟随人物威亚上升/下降，速度0.5米/秒 |
| 升格环绕 | 升格4倍速下，镜头以人物为圆心半径2米环绕360度 |
| 急推特写 | 法术爆发瞬间急推特写，速度>2米/秒 |
| 缓拉大景 | 从中景缓拉至大远景展示仙山全貌，速度0.5米/秒 |

**禁用运镜方式**：手持晃动（破坏仙气感）、低调暗光（破坏出尘感）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |

| 承 | 激战爆发 | 升格环绕、急推特写、法术光晕扩散 |
| 转 | 归隐落幕 | 缓拉大景、人物背影远去、云海翻涌 |

## 8. 典型镜头模板

### 模板 G07-T1 — 出尘登场

```
镜头ID：场号-01
景别：中景 → 全景
运镜：镜头机位固定于人物正前方3米处，焦段自50mm匀速变至135mm长焦，从中景压缩到全景，时长6秒
场景地点：S编号 仙山云海顶部外，云海地面，仙气漫射光照明
环境光影：仙气漫射光自四周平射色温7000K冷白，高调明亮，云海在脚下翻涌产生动态光斑
出场人物：C001 仙门弟子（云海中央悬浮站立，白色道袍衣袂飘飘）
人物动作：C001 双手负于身后，衣袂随风飘动，发丝轻扬；眼神平静俯瞰云海下方
面部表情：C001 眉眼平直，嘴角微压，下颌微抬（情绪：出尘俯瞰）
情绪基调：出尘登场
台词/音效：无台词 / 音效：仙鹤鸣叫、云海翻涌声、配乐古琴主旋律渐起
转场方式：切
```

### 模板 G07-T2 — 激战升格

```
镜头ID：场号-05
景别：中景
运镜：镜头以C001为圆心，半径2米，围绕C001匀速环绕180度，升格4倍速，从中景过渡到近景，时长6秒（实际1.5秒）
场景地点：S编号 仙门演武场上空外，云海地面，法术彩光+仙气漫射光照明
环境光影：法术紫红光自C002掌心扩散色温2500K暖红，仙气冷白光自四周平射色温7000K，高调明亮+局部紫红斑
出场人物：C001 仙门弟子（云海中央悬浮，右手持飞剑）；C002 魔修（C001对侧10米处，双手聚紫红法术球）
人物动作：C001 右手挥剑，飞剑脱手化作青蓝剑气直冲C002；C002 双手紫红法术球骤然爆发迎击剑气
面部表情：C001 眉头紧锁，眼神锐利如刀，下颌发力（情绪：激战专注）；C002 眉头深锁，嘴角扭曲上扬（情绪：狂暴施法）
情绪基调：激战爆发
台词/音效：C001："破！"（语气凌厉爆发） / 音效：剑鸣嗡嗡、法术爆裂轰鸣、配乐编钟重音炸响
转场方式：白闪
```

### 模板 G07-T3 — 归隐落幕

```
镜头ID：场号-12
景别：中景 → 全景
运镜：镜头从人物正前方1.5米处中近景，沿水平方向缓慢后拉至5米处大远景，速度0.5米/秒，时长8秒
场景地点：S编号 仙山归隐亭外，青石板地面，夕阳金光照明
环境光影：夕阳金光自西方平射色温2500K暖橙，仙气冷白光自四周柔射色温7000K，高调明亮，云海被夕阳染成金红色
出场人物：C001 仙门弟子（青石板上背对镜头缓步远去，道袍后摆拖地）
人物动作：C001 双手负于身后缓步前行，每一步衣袂飘动；逐渐走远消失在云海中
面部表情：不可见（背影）
情绪基调：归隐落幕
台词/音效：无台词 / 音效：远处钟磬三响、配乐古琴主旋律渐弱收尾
转场方式：叠化
```
