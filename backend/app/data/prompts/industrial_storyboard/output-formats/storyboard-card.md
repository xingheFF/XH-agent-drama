# 分镜卡 Markdown 输出模板

> 短剧分镜工业化生产的标准输出格式之一。每条镜头输出一张 Markdown 表格分镜卡，供摄影师/导演拍摄使用。

---

## 一、单镜头分镜卡模板

每条镜头按以下 Markdown 格式输出：

```markdown
## 镜头 <场号>-<镜号>

| 字段 | 内容 |
|---|---|
| 镜头ID | <场号>-<镜号> |
| 时间码 | 第X秒 - 第Y秒 |
| 景别 | <近景/中近景/中景/特写> |
| 运镜 | <精确起止状态+运动轨迹+速度描述，禁用"运镜"二字> |
| 场景地点 | <S编号 场所名内/外，地面材质，照明方式> |
| 环境光影 | <光源方向+色温+明暗氛围+动态光影效果> |
| 出场人物 | <C编号 角色名（站位描述）；多人物用分号> |
| 人物动作 | <精确到肢体部位，与台词节奏同步> |
| 面部表情 | <精确到肌肉状态，对应情绪> |
| 情绪基调 | <本场镜头的核心情绪，2-4字短语> |
| 台词/音效 | <说话人："台词"（语气描述）/ 音效：音效描述> |
| 转场方式 | <切/叠化/闪回/运镜转场/黑场/白闪> |
```

## 三、多镜头组分镜卡模板

当一组连续镜头属于同一场景同一情节段时，按以下格式输出：

```markdown
## 场景 <S编号> — <情节段名>

> 场景：S<编号> <场景名>
> 题材风格包：G<编号> <题材名>
> 情绪曲线：<起 - 承 - 转 - 合>

### 镜头 <场号>-<镜号01>

| 字段 | 内容 |
|---|---|
| ... | ... |

### 镜头 <场号>-<镜号02>

| 字段 | 内容 |
|---|---|
| ... | ... |

### 镜头 <场号>-<镜号03>

| 字段 | 内容 |
|---|---|
| ... | ... |
```

## 五、输出规范

### 5.1 强制要求

| 要求 | 说明 |
|---|---|
| 12 字段完整 | 每张分镜卡必须包含全部 12 个字段，缺一不可 |
| 运镜禁词 | "运镜"字段中严禁出现"运镜"二字 |
| 角色ID 体系 | 出场人物字段必须使用 `C编号` 体系 |
| 场景ID 体系 | 场景地点字段必须使用 `S编号` 体系 |
| 情绪基调 | 必须为 2-4 字短语，对应风格包的情绪节奏阶段 |
| 时间码累计 | 时间码按全剧时间轴累计计算，非每场独立 |

### 5.2 可选扩展

| 扩展字段 | 用途 | 何时使用 |
|---|---|---|
| 道具特写 | 列出本镜需特写的关键道具 | 关键道具首次出现时 |
| 群演描述 | 描述背景群演的动作与服装 | 群戏场景 |
| 站位图 | 文字版俯视站位图 | 3 人以上复杂群戏 |
| 配乐节点 | 标注配乐起伏节点 | 配乐紧密配合剧情时 |

### 5.3 扩展字段示例

在分镜卡表格末尾追加扩展字段：

```markdown
| 站位图 | 案台 ─── 沈砚(C001) / │ / 李掌柜(C002) / │ / 堂下围观众人 |
| 道具特写 | 账册（红木封面，铜扣） |
| 群演描述 | 堂下站立4名衙役，统一青色衙役服，手持水火棍垂立 |
| 配乐节点 | C001拍案瞬间配乐低频重音炸响，持续至镜头结束 |
```

## 六、质检清单

输出分镜卡前自检以下 8 项：

- [ ] 12 字段是否全部填写？
- [ ] 运镜字段是否避免了"运镜"二字？
- [ ] 场景地点是否使用了 `S编号` 体系？
- [ ] 出场人物是否使用了 `C编号` 体系？
- [ ] 面部表情是否精确到肌肉状态？
- [ ] 人物动作是否标注了与台词的节奏关系？
- [ ] 情绪基调是否为 2-4 字短语？
- [ ] 时间码是否按全剧累计计算？
