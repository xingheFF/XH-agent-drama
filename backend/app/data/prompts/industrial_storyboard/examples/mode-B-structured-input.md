# 模式 B 示例：结构化镜头清单

> 用户提供 CSV/JSON/Markdown 表格形式的镜头清单（含场号、镜号、景别、台词、出场人物等原始字段），技能直接套公式补全 12 字段，生成双版本输出。

---

## 一、用户输入

```
[分镜生成] 题材：现代复仇，模式B

镜头清单：

| 场号 | 镜号 | 景别 | 出场人物 | 台词 | 备注 |
|---|---|---|---|---|---|
| 001 | 01 | 中景 | 林深 | 无 | 雨夜街角登场 |
| 001 | 02 | 特写 | 林深 | 无 | 烟头明灭 |
| 001 | 03 | 中景 | 林深、苏晚 | "三年了，你终于知道怕了？" | 推门进入包厢对峙 |
| 001 | 04 | 近景 | 苏晚 | 无 | 红酒杯落地 |
| 001 | 05 | 中景 | 林深 | 无 | 转身离去背影 |

角色档案：
- C001 林深：30岁，黑色长款风衣，撑黑色长柄伞，眉眼锐利，隐忍压抑
- C002 苏晚：28岁，红色晚礼服，长发披肩，妆容精致，眼神闪烁

场景档案：
- S001 雨夜街角外：湿滑石板路，路灯+霓虹灯照明，色温混合3000K+6000K
- S002 高端会所包厢内：厚地毯，水晶吊灯，色温4000K暖白
```

## 二、AI 内部执行流程

### Step 1：解析输入 → 判断模式 B

- 用户提供了结构化镜头清单（Markdown 表格）+ 角色档案 + 场景档案 → 模式 B
- 题材：现代复仇 → 加载风格包 `G02`

### Step 2：读取角色档案

直接调用用户提供的角色档案：

| C编号 | 角色名 | 外貌/服装简述 |
|---|---|---|
| C001 | 林深 | 30岁，黑色长款风衣，撑黑色长柄伞，眉眼锐利，隐忍压抑 |
| C002 | 苏晚 | 28岁，红色晚礼服，长发披肩，妆容精致，眼神闪烁 |

### Step 3：读取场景档案

直接调用用户提供的场景档案：

| S编号 | 场景名 | 空间属性 |
|---|---|---|
| S001 | 雨夜街角外 | 湿滑石板路，路灯+霓虹灯照明，色温混合3000K+6000K |
| S002 | 高端会所包厢内 | 厚地毯，水晶吊灯，色温4000K暖白 |

### Step 4：套用 12 字段公式逐镜生成

根据用户提供的"备注"列判断情绪阶段，根据风格包 G02 加载光影/运镜/情绪基调参数，逐镜补全 12 字段。

### Step 5：运镜禁词校验

扫描所有镜头的"运镜"字段，确认无"运镜"二字。

### Step 6：双版本输出

每条镜头同时输出分镜卡 + AI 模型提示词块。

## 四、模式 B 与模式 A 的差异

| 维度 | 模式 A（剧本自动拆分） | 模式 B（结构化镜头清单） |
|---|---|---|
| 输入格式 | 剧本文字段落 | Markdown 表格 / CSV / JSON |
| 镜头切分 | AI 自动按"动作变化+台词变化"切分 | 用户预先在表格中指定镜号 |
| 角色档案 | AI 自动从剧本抽取并创建 | 用户可直接提供档案，AI 调用 |
| 场景档案 | AI 自动从剧本抽取并创建 | 用户可直接提供档案，AI 调用 |
| 情绪阶段 | AI 根据剧本动词/形容词推断 | 用户在"备注"列标注，AI 采用 |
| 适用场景 | 剧本初稿快速拆分 | 已有分镜表，需补全字段 |
