# 12 字段分镜公式定义

> 短剧分镜工业化生产的标准字段公式。每条镜头必须完整填写 12 个字段，缺一不可。

---

## 一、字段总表

| 序号 | 字段名 | 格式要求 | 示例 |
|---|---|---|---|
| 1 | 镜头ID | 场号(3位)-镜头号(2位) | `002-05` |
| 2 | 时间码 | 第X秒 - 第Y秒 | `第32秒 - 第35秒` |

| 5 | 场景地点 | `S编号 场所内/外，地面材质，照明方式` | `S001 县衙大堂内，青石板地面，烛火照明` |

## 二、字段详细规范

### 字段 1 — 镜头ID

- 格式：`场号-镜头号`
- 场号：3 位数字，从 `001` 起（例：`001`、`002`、`010`）
- 镜头号：2 位数字，从 `01` 起，每场独立编号（例：`01`、`05`、`12`）
- 组合示例：`001-01`、`002-05`、`010-12`

### 字段 2 — 时间码

- 格式：`第X秒 - 第Y秒`
- X 为镜头起始秒数，Y 为镜头结束秒数
- 全剧时间轴累计计算，非每场独立
- 单镜头时长建议 2-6 秒，关键镜头可延长至 8-10 秒

### 字段 3 — 景别

| 景别 | 取景范围 | 适用场景 |
|---|---|---|
| 特写 | 单一局部（眼/手/物） | 情绪爆发点、关键道具 |
| 近景 | 胸部以上 | 对话、表情刻画 |
| 中近景 | 腰部以上 | 半身动作、对话 |
| 中景 | 膝盖以上 | 全身动作、多人站位 |
| 全景 | 整体环境+人物 | 场景交代、空间关系 |
| 远景 | 大环境 | 气氛渲染、转场过渡 |

### 字段 4 — 运镜

- **禁用"运镜"二字**（详见 `forbidden-words.md`）
- 必须包含三要素：**起止状态** + **运动轨迹** + **速度描述**
- 起止状态：起始构图（景别+机位）→ 结束构图（景别+机位）
- 运动轨迹：水平/垂直/弧线/螺旋/环绕/跟拍路径
- 速度描述：匀速/加速/减速/急推/缓拉

**示例**：
- ✅ `镜头从人物肩后2米处沿水平方向匀速推进至0.5米近景`
- ✅ `镜头自人物脚下1米处垂直上升至头顶上方0.5米处，速度渐缓`
- ✅ `镜头在人物正前方1.5米处保持固定，仅左右微调0.2米模拟呼吸感`
- ❌ `运镜：推进`（使用了禁词）
- ❌ `镜头推进`（缺少起止状态和轨迹）

### 字段 5 — 场景地点

- 格式：`S编号 场所名内/外，地面/地表材质，主要照明方式`
- 场景ID：`S001`、`S002`……（按出场顺序）
- 场所名：具体空间名（例：县衙大堂、雨夜巷口、CEO 办公室）
- 内外景：内 / 外 / 内外过渡
- 地面材质：青石板 / 木地板 / 水泥地 / 草地 / 黄沙
- 照明方式：烛火 / 日光 / 霓虹灯 / 月光 / 篝火

**示例**：
- `S001 县衙大堂内，青石板地面，烛火照明`
- `S005 雨夜巷口外，湿滑石板路，路灯+车灯照明`
- `S012 CEO 办公室内，深色木地板，落地窗日光`

### 字段 6 — 环境光影

- 必填四要素：**光源方向** + **色温** + **明暗氛围** + **动态光影效果**
- 光源方向：顶光 / 侧光 / 逆光 / 顺光 / 底光 / 多向光
- 色温：以 Kelvin 值+主观色调描述（例：`2700K 暖橙`、`5600K 冷白`、`混合色温`）
- 明暗氛围：高调明亮 / 低调昏暗 / 高反差 / 柔光均匀
- 动态光影效果：无 / 烛火摇曳阴影 / 雨水反光闪烁 / 车灯扫过 / 树影婆娑

**示例**：
- `顶光自梁下垂直压下，色温2700K暖橙，明暗对比强烈，烛火摇曳产生动态阴影`
- `逆光来自巷口路灯，色温3200K冷白，低调昏暗，雨水反光在地面闪烁`
- `侧光来自落地窗，色温5600K日光白，高反差，云影移动产生明暗变化`

### 字段 7 — 出场人物

- 格式：`C编号 角色名（站位描述）`
- 多人物用分号 `;` 分隔
- 站位描述包含：相对位置（前景/中景/后景）+ 朝向 + 与关键参照物的距离

**示例**：
- `C001 沈砚（案台后正中站立，面朝镜头偏左30度）`
- `C001 沈砚（案台后正中站立）；C002 李掌柜（案台前偏左，距案台1米）`
- `C003 林深（前景，背对镜头）；C004 苏晚（中景，侧身面对林深，相距2米）`

### 字段 8 — 人物动作

- 精确到**肢体部位**（右手/左手/双肩/腰部/脚步/眼神方向）
- 与**台词节奏同步**：动作要在台词之前/之中/之后明确标注
- 多人物动作用分号分隔，按时间顺序排列

**示例**：
- `C001 右手猛地将账册拍在案台上（台词前0.5秒），身体微微前倾，左手按住案台边缘`
- `C002 后退半步（同步于C001拍案动作），双手在身前微微发抖，嘴唇开合两次才出声`

### 字段 9 — 面部表情

- 精确到**肌肉状态**（眉头/眼睑/瞳孔/嘴角/下颌/面颊/鼻翼）
- 对应**情绪标签**（愤怒/惊恐/悲伤/冷酷/犹豫/坚定）
- 多人物表情用分号分隔

**示例**：
- `C001 眉头紧锁，眼神锐利如刀，嘴角紧绷下压，下颌发力，面颊肌肉微颤（情绪：压抑的怒火）`
- `C002 脸色刷白，瞳孔微缩，上下唇分离颤抖，下颌微张（情绪：惊恐）`

### 字段 10 — 情绪基调

- 2-4 字短语，描述本场镜头的**核心情绪**
- 一场戏可有多个情绪转折，每个镜头标注当前所处情绪阶段
- 与风格包的"情绪节奏"参数呼应

**示例**：`压迫对峙` / `震惊反转` / `温情舒缓` / `危机暗涌` / `决意赴死` / `甜蜜心动`

### 字段 11 — 台词/音效

- 格式：`说话人："台词内容" / 音效：音效描述`
- 说话人用 `C编号` 或角色名
- 台词若有停顿、重音、语气变化，需在台词后用括号标注
- 音效按重要性排序，最多列 3 个关键音效
- 若本镜头无台词，标注 `无台词`

**示例**：
- `C001："这三千两银子，你作何解释？"（语气冰冷，重音在"解释"） / 音效：账册拍案重击声、烛火噼啪声、远处更鼓声`
- `无台词 / 音效：雨声密集、脚步急促、远处雷声轰鸣`

### 字段 12 — 转场方式

| 转场方式 | 适用场景 |
|---|---|
| 切 | 默认，节奏紧凑 |
| 叠化 | 时间流逝、回忆过渡 |
| 闪回 | 突然回忆、真相揭露 |
| 运镜转场 | 通过镜头运动无缝衔接下一镜（例：推向物体黑场切入下镜） |
| 黑场 | 段落结束、情绪留白 |
| 白闪 | 震惊、爆炸、关键瞬间 |

**示例**：`切` / `叠化` / `闪回（黑白处理）` / `运镜转场（推向账册特写后切入下镜）`

## 三、字段填充优先级

当信息不完整时，按以下优先级填充：

1. **景别** — 根据台词重要性判断（关键台词→近景/特写）
2. **情绪基调** — 从剧本动词和形容词抽取
3. **人物动作** — 从剧本动词抽取
4. **面部表情** — 从剧本形容词+情绪基调反推
5. **环境光影** — 从风格包参数加载
6. **运镜** — 从风格包运镜偏好+景别推导
7. **转场方式** — 默认"切"，情绪转折处改为对应转场
8. **时间码** — 根据镜头动作复杂度估算（简单动作2-3秒，复杂动作4-6秒）
