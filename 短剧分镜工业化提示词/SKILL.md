# SKILL: 短剧分镜工业化提示词生成器

> **Skill ID**: `storyboard-prompt`
> **Version**: 1.0
> **触发词**: `分镜生成` / `生成分镜` / `分镜提示词`
> **入口文件**: 本文件（SKILL.md）

---

## 一、技能描述

本技能用于短剧分镜脚本的工业化生产。用户提供剧本片段、结构化镜头清单或单镜描述，技能自动输出**标准 12 字段分镜卡（Markdown 表格）** 与 **AI 模型中文提示词块** 双版本，满足"喂 AI 绘图/视频模型"与"给摄影师拍摄脚本"两种用途。

---

## 二、输入模式

### 模式 A — 剧本自动拆分

用户提供一段剧本文字，技能自动按"场号-镜头号"切分为多个镜头单元，逐镜套用 12 字段公式。

**输入示例**：
```
[分镜生成] 题材：古装权谋
剧本片段：
"沈砚将账册重重拍在案台上，烛火被震得摇曳。李掌柜脸色刷白，后退半步。沈砚冷声开口：'这三千两银子，你作何解释？'"
```

### 模式 B — 结构化镜头清单

用户提供 CSV/JSON/Markdown 表格形式的镜头清单（含场号、镜号、景别、台词、出场人物等原始字段），技能直接套公式补全 12 字段。

**输入示例**：
```
[分镜生成] 题材：现代复仇，模式B
镜头清单：
| 场号 | 镜号 | 景别 | 台词 | 出场人物 |
| 002  | 05   | 近景 | "你还有脸回来？" | 林深、苏晚 |
```

### 模式 C — 单镜直填

用户描述一个镜头的核心信息，技能生成单张分镜卡。

**输入示例**：
```
[分镜生成] 题材：悬疑罪案
单镜：C004 雨夜巷口，嫌疑人回头，惊恐表情，慢推镜头
```

## 四、调用规则

### 4.1 触发语法

```
[分镜生成] 题材：<题材名> [, 模式：<A/B/C>]
<剧本片段 / 镜头清单 / 单镜描述>
```

- `题材名` 必须匹配 `style-packs/index.md` 中的 14 套风格包之一（可用中文题材名或 G01-G14 编号）。
- `模式` 可选，未指定时由技能自动识别输入格式。

### 4.2 角色档案调用

- 若用户已提供角色档案（`C001` 体系），技能优先调用档案中的外貌/服装/性格标签/口头动作。
- 若未提供，技能从剧本中自动抽取角色并分配 ID（按出场顺序 `C001`、`C002`……）。
- 角色档案模板见 `templates/character-template.md`。

### 4.3 场景档案调用

- 若用户已提供场景档案（`S001` 体系），技能优先调用档案中的空间属性+环境属性。
- 若未提供，技能从剧本中自动抽取场景并分配 ID（按出场顺序 `S001`、`S002`……）。
- 场景档案模板见 `templates/scene-template.md`。

### 4.4 风格包加载

- 技能根据用户指定的题材，加载对应风格包的完整参数（光影基调/运镜偏好/情绪节奏/转场偏好/色彩特征/构图特征）。
- 用户可临时覆盖某项参数（例：`题材：古装权谋，但光影改为冷蓝调`）。
- 风格包索引见 `style-packs/index.md`，各包详情见 `style-packs/<pack-name>.md`。

### 4.5 运镜禁词校验（强制）

- 输出的"运镜"字段中，**严禁出现"运镜"二字**。
- 必须改写为**精确起止状态 + 运动轨迹**描述。
- 改写模板与规则见 `formula/forbidden-words.md`。

### 4.6 输出格式（强制）

每条镜头必须**同时输出两个版本**：

**版本 1 — 分镜卡（Markdown 表格）**

参考 `output-formats/storyboard-card.md`，格式如下：

```markdown
## 镜头 002-05

| 字段 | 内容 |
|---|---|
| 镜头ID | 002-05 |
| 时间码 | 第32秒 - 第35秒 |
| 景别 | 近景 |
| 运镜 | 镜头从人物肩后2米处沿水平方向匀速推进至0.5米近景 |
| 场景地点 | S001 县衙大堂内，青石板地面，烛火照明 |
| 环境光影 | 顶光自梁下垂直压下，色温2700K暖橙，明暗对比强烈，烛火摇曳产生动态阴影 |
| 出场人物 | C001 沈砚（案台后正中站立）、C002 李掌柜（案台前偏左，距案台1米） |
| 人物动作 | C001 右手猛地将账册拍在案台上，身体微微前倾；C002 后退半步，双手微微发抖 |
| 面部表情 | C001 眉头紧锁，眼神锐利，嘴角紧绷，下颌发力；C002 脸色刷白，瞳孔微缩 |
| 情绪基调 | 压迫对峙 |
| 台词/音效 | C001："这三千两银子，你作何解释？" / 音效：账册拍案重击声、烛火噼啪声 |
| 转场方式 | 切 |
```

**版本 2 — AI 模型中文提示词块**

参考 `output-formats/model-prompt-cn.md`，格式如下：

```
【镜头002-05 提示词】
场景：县衙大堂内，青石板地面，烛火照明，顶光自梁下垂直压下，色温2700K暖橙，明暗对比强烈，烛火摇曳产生动态阴影。
人物：沈砚（C001，青色官袍，束发玉冠，案台后正中站立），李掌柜（C002，灰布长衫，案台前偏左，距案台1米）。
动作：沈砚右手猛地将账册拍在案台上，身体微微前倾；李掌柜后退半步，双手微微发抖。
表情：沈砚眉头紧锁，眼神锐利，嘴角紧绷，下颌发力；李掌柜脸色刷白，瞳孔微缩。
镜头：从人物肩后2米处沿水平方向匀速推进至0.5米近景，近景，时长3秒。
情绪：压迫对峙。
转场：切。
```

## 六、错误处理

| 异常场景 | 处理方式 |
|---|---|
| 用户未指定题材 | 提示用户从 14 套风格包中选择，并列出题材清单 |
| 用户指定的题材不在 14 套之内 | 提示用户选择最接近的内置题材，或让用户自定义光影/运镜/情绪基调参数 |
| 剧本片段无法识别镜头边界 | 按句号/感叹号/问号切分，每个完整句作为一个镜头单元 |
| 角色档案冲突（同名不同 ID） | 以用户最新提供的档案为准，并提示冲突 |
| 输出中出现"运镜"二字 | 立即重写该字段，改用精确轨迹描述 |
