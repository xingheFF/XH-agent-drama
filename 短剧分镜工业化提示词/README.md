# 短剧分镜工业化提示词技能包

> 一套面向短剧工业化生产的分镜提示词生成体系。输入剧本或结构化镜头清单，自动输出标准 12 字段分镜卡 + AI 模型中文提示词双版本，满足"喂 AI 绘图/视频模型"与"给摄影师拍摄脚本"两种用途。

---

## 一、技能包定位

| 维度 | 说明 |
|---|---|
| **适用场景** | 短剧分镜脚本撰写、AI 视频生成提示词构建、拍摄现场脚本输出 |
| **输入双模式** | A. 剧本自动拆分 / B. 结构化镜头清单 / C. 单镜直填 |
| **输出双用途** | 1. 分镜卡（Markdown 表格，供摄影师） / 2. AI 模型中文提示词块（供即墨/可灵/Runway 等） |
| **角色体系** | `C001` 格式，可预设档案或自动抽取 |
| **场景体系** | `S001` 格式，可预设档案或自动抽取 |
| **题材风格包** | 内置 14 套（古装权谋/现代复仇/甜宠/悬疑罪案/都市职场/民国谍战/仙侠玄幻/校园青春/家庭伦理/战争军旅/惊悚灵异/电竞热血/大漠武侠/宫廷古装） |
| **运镜禁词校验** | 自动拦截"运镜"二字，改写为精确轨迹描述 |

## 三、快速使用

### 调用方式

向 AI 输入以下任一指令即可触发本技能：

```
[分镜生成] 题材：古装权谋
剧本片段：
"沈砚将账册重重拍在案台上，烛火被震得摇曳。李掌柜脸色刷白，后退半步。"
请自动拆分镜头并生成分镜卡。
```

或结构化输入：

```
[分镜生成] 题材：现代复仇，模式B
镜头清单：
| 场号 | 镜号 | 景别 | 台词 | 出场人物 |
| 002  | 05   | 近景 | "你还有脸回来？" | 林深、苏晚 |
```

### 工作流（AI 内部执行顺序）

```
1. 解析输入 → 判断模式 A/B/C
2. 读取/创建角色档案（C001 体系）+ 场景档案（S001 体系）
3. 匹配题材风格包（G01-G14）→ 加载光影/运镜/情绪基调参数
4. 按 12 字段公式逐项填充
5. 运镜禁词校验 → 自动改写为精确轨迹描述
6. 双版本输出：
   ├─ 分镜卡（Markdown 表格）
   └─ AI 模型中文提示词块
```

## 四、12 字段公式速查

| 序号 | 字段 | 格式要求 |
|---|---|---|
| 1 | 镜头ID | 场号-镜头号（例：`002-05`） |
| 2 | 时间码 | 第X秒 - 第Y秒 |
| 3 | 景别 | 近景/中近景/中景/特写 |
| 4 | 运镜 | 精确描述起止状态+运动轨迹，**禁用"运镜"二字** |
| 5 | 场景地点 | 精确到具体空间+环境属性（例：`S001 县衙大堂内，青石板地面，烛火照明`） |
| 6 | 环境光影 | 光源方向+色温+明暗氛围+动态光影效果 |
| 7 | 出场人物 | 角色ID+站位关系 |
| 8 | 人物动作 | 精确到肢体部位，与台词节奏同步 |
| 9 | 面部表情 | 精确到肌肉状态，对应情绪 |
| 10 | 情绪基调 | 本场镜头的核心情绪 |
| 11 | 台词/音效 | 对应台词内容+关键音效，标注说话人 |
| 12 | 转场方式 | 切/叠化/闪回/运镜转场 |

详细公式与示例见 `formula/12-fields-formula.md`。
