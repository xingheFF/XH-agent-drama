# 运镜禁词校验规则

> 输出的"运镜"字段中，**严禁出现"运镜"二字**。必须改写为**精确起止状态 + 运动轨迹 + 速度描述**的完整描述。

---

## 一、禁词清单

以下词汇在"运镜"字段中**禁止使用**：

| 禁词 | 禁用原因 |
|---|---|
| 运镜 | 模糊笼统，未描述具体运动 |
| 镜头运动 | 同上 |
| 推 | 缺少起止距离与速度 |
| 拉 | 同上 |
| 摇 | 缺少轴向、角度、速度 |
| 移 | 缺少方向、距离、速度 |
| 跟 | 缺少跟拍对象、相对位置、速度 |
| 升 / 降 | 缺少起止高度、速度 |
| 甩 | 缺少起止构图、方向、速度 |
| 环绕 | 缺少环绕中心、半径、速度 |
| 变焦 | 缺少焦段变化范围、速度 |
| 手持 | 仅描述稳定方式，未描述运动 |

> **注**：以上词汇**可以组合使用**于完整描述中，但不能**单独使用**作为运镜字段的全部内容。

---

## 二、合规三要素

每条运镜描述必须包含以下三要素：

### 要素 1 — 起止状态

- **起始构图**：景别 + 机位（相对主体的位置）
- **结束构图**：景别 + 机位

示例：`从人物肩后2米处中景` → `至0.5米处近景`

### 要素 2 — 运动轨迹

| 轨迹类型 | 描述方式 |
|---|---|
| 水平直线 | 沿水平方向 / 沿X轴方向 |
| 垂直直线 | 垂直上升 / 垂直下降 |
| 弧线 | 以主体为圆心，半径R米，做弧线运动 |
| 螺旋 | 螺旋下降/上升，半径渐变 |
| 环绕 | 围绕主体做圆周运动，半径R米 |
| 跟拍 | 在主体侧前方/侧后方平行移动，保持距离R米 |
| 固定微调 | 机位固定，仅做0.1-0.3米的微调 |

示例：`沿水平方向匀速推进`

### 要素 3 — 速度描述

| 速度标签 | 描述 |
|---|---|
| 静止 | 机位完全不动（固定镜头） |
| 微动 | 仅0.1-0.3米微调，模拟呼吸感 |
| 缓慢 | 0.3-0.5米/秒 |
| 匀速 | 0.5-1.0米/秒 |
| 快速 | 1.0-2.0米/秒 |
| 急推/急拉 | >2.0米/秒，强调冲击感 |
| 渐进/渐离 | 速度由慢变快或由快变慢 |

示例：`匀速推进` / `急推进` / `渐进式推进`

## 三、改写模板

### 模板 1 — 简单推镜

❌ 错误：`推`

✅ 正确：`镜头从人物正前方2米处中景，沿水平方向匀速推进至0.5米处近景`

### 模板 2 — 简单拉镜

❌ 错误：`拉`

✅ 正确：`镜头从人物正面0.5米处近景，沿水平方向缓慢后拉至3米处全景`

### 模板 3 — 摇镜

❌ 错误：`摇`

✅ 正确：`镜头机位固定于人物侧方1.5米处，水平方向自左向右匀速旋转60度，从中景摇至中近景`

### 模板 4 — 升降镜

❌ 错误：`升`

✅ 正确：`镜头自人物脚下0.5米处垂直匀速上升至头顶上方0.3米处，从中近景过渡到俯拍全景`

### 模板 5 — 环绕镜头

❌ 错误：`环绕`

✅ 正确：`镜头以人物为圆心，保持1.5米半径，围绕人物匀速环绕180度，从中景过渡到近景`

### 模板 6 — 跟拍镜头

❌ 错误：`跟`

✅ 正确：`镜头在人物侧前方1米处，与人物保持同步匀速前行（速度约0.8米/秒），从中近景维持到中近景`

### 模板 7 — 甩镜

❌ 错误：`甩`

✅ 正确：`镜头从主体A正前方1.5米处中近景，水平方向急速甩向主体B，落幅于主体B侧方1米处中景`

### 模板 8 — 变焦推拉

❌ 错误：`变焦`

✅ 正确：`镜头机位固定于人物正前方3米处，焦段自35mm匀速变至85mm，景别从中景过渡到特写`

### 模板 9 — 手持晃动

❌ 错误：`手持`

✅ 正确：`镜头以手持方式在人物侧前方1-1.5米范围内做不规则微动（幅度0.1-0.2米，频率2-3Hz），模拟紧张感`

### 模板 10 — 固定微调

❌ 错误：`固定`

✅ 正确：`镜头机位固定于人物正前方1.5米处，仅做0.2米幅度的左右微调，模拟呼吸感，景别维持中近景不变`

## 四、校验流程

```
Step 1: 扫描输出中"运镜"字段
        ↓
Step 2: 检查是否出现禁词（运镜/推/拉/摇/移/跟/升/降/甩/环绕/变焦/手持）
        ├─ 无禁词 → 通过
        └─ 有禁词 → Step 3
        ↓
Step 3: 判断是否为完整描述（包含起止状态+运动轨迹+速度描述）
        ├─ 是完整描述 → 通过（禁词被嵌入在完整描述中属合规）
        └─ 仅用禁词作为全部内容 → Step 4
        ↓
Step 4: 根据禁词类型匹配改写模板，重写运镜字段
        ↓
Step 5: 二次扫描确认无违规
```

## 五、特殊场景

### 场景 1 — 一镜到底长镜头

描述格式：`镜头从[起始构图]开始，先[运动段1描述]，到达[中间构图1]后[运动段2描述]，最终落幅于[结束构图]`

示例：`镜头从人物A正前方2米处中景开始，沿水平方向匀速推进至0.8米处近景（人物A说话），随后水平右移至人物B正前方0.8米处近景（人物B回应），最终缓慢后拉至3米处双人中景`

### 场景 2 — 升格（慢动作）镜头

需在运镜描述末尾标注 `(升格，X倍速)` 或 `(降格，X倍速)`

示例：`镜头从人物侧方1.5米处中近景，垂直匀速上升至头顶上方0.3米处 (升格，4倍速)`

### 场景 3 — 主观视角镜头

描述格式：`主观视角（C编号 角色名），[运动轨迹描述]`

示例：`主观视角（C002 李掌柜），镜头从平视角度缓慢上抬30度，最终落幅于C001面部特写`

### 场景 4 — 蒙太奇快切

每个子镜头单独描述，用 `→` 串联

示例：`镜头1：C001右手拍案特写（固定）→ 镜头2：C002瞳孔收缩特写（急推0.3米）→ 镜头3：烛火被震摇曳近景（微动）`

## 六、自检清单

输出"运镜"字段前自检：

- [ ] 是否出现了"运镜"二字？（应为否）
- [ ] 是否包含**起始构图**（景别+机位）？
- [ ] 是否包含**结束构图**（景别+机位）？
- [ ] 是否包含**运动轨迹**（方向+路径）？
- [ ] 是否包含**速度描述**（匀速/急推/缓慢等）？
- [ ] 升格/降格镜头是否标注了倍速？
- [ ] 主观视角是否标注了角色？
