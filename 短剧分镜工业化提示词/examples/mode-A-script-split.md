# 模式 A 示例：剧本自动拆分

> 输入一段剧本，技能自动按"场号-镜头号"切分多个镜头单元，逐镜套用 12 字段公式生成双版本输出。

---

## 一、用户输入

```
[分镜生成] 题材：古装权谋

剧本片段：
"深夜，县衙大堂。沈砚端坐案后，烛火摇曳。李掌柜跪在堂下，浑身发抖。
沈砚将账册重重拍在案台上，烛火被震得摇曳。李掌柜脸色刷白，后退半步。
沈砚冷声开口：'这三千两银子，你作何解释？'
李掌柜扑通跪下，额头磕在青石板上：'大人明鉴，小的不知啊！'
沈砚起身，绕过案台，缓步走向李掌柜。每一步皮靴踏地声清晰。
沈砚俯身，右手食指挑起李掌柜的下颌：'再编，本官就让你尝尝夹棍的滋味。'"
```

## 二、AI 内部执行流程

### Step 1：解析输入 → 判断模式 A

- 用户提供了剧本片段 → 模式 A（剧本自动拆分）
- 题材：古装权谋 → 加载风格包 `G01`

### Step 2：读取/创建角色档案

从剧本中识别出场人物，分配 `C编号`：

| C编号 | 角色名 | 外貌/服装简述 |
|---|---|---|
| C001 | 沈砚 | 青色官袍，束发玉冠，30岁，眉眼锐利 |
| C002 | 李掌柜 | 灰布长衫，50岁，身材瘦削，面有惧色 |

### Step 3：读取/创建场景档案

从剧本中识别场景，分配 `S编号`：

| S编号 | 场景名 | 空间属性 |
|---|---|---|
| S001 | 县衙大堂内 | 长12米×宽8米×高4米，青石板地面，烛火照明 |

### Step 4：剧本切分为镜头单元

按"动作变化+台词变化"切分，识别 5 个镜头单元：

| 镜头ID | 切分依据 | 情绪阶段 |
|---|---|---|
| 002-01 | 场景交代+人物登场 | 压抑沉寂 |
| 002-02 | 沈砚拍案+李掌柜受惊 | 暗流涌动 |
| 002-03 | 沈砚质问台词 | 压迫对峙 |
| 002-04 | 李掌柜磕头求饶 | 惊恐求饶 |
| 002-05 | 沈砚起身逼近 | 威权碾压 |

### Step 5：套用 12 字段公式逐镜生成

（详见下文"三、双版本输出"）

### Step 6：运镜禁词校验

扫描所有镜头的"运镜"字段，确认无"运镜"二字。

### Step 7：双版本输出

每条镜头同时输出分镜卡 + AI 模型提示词块。
