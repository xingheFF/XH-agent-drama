# G01 古装权谋

> 朝堂官场、权谋对决题材。以压抑的顶光、烛火暖橙、对称构图营造仪式感与压迫感。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 顶光自梁下垂直压下 |
| 色温 | 2700K 暖橙为主，局部5600K冷白对比 |
| 明暗氛围 | 高反差低调光，暗部占画面60%以上 |
| 动态光影效果 | 烛火摇曳产生动态阴影、烟雾中光柱可见 |

**辅助光源**：
- 烛火点光源（多盏分布，色温2000-2500K）
- 窗格自然光侧入（色温5500K，弱）
- 反光板补面部阴影（强度低于主光1档）

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 缓推 | 从中景2-3米匀速推进至近景0.5-0.8米，速度0.3-0.5米/秒，用于台词落幅 |
| 对称定镜 | 机位正中、水平居中，人物处于黄金分割点，用于权谋对峙 |
| 缓慢横移 | 沿水平方向0.2-0.3米/秒横移，展示人物群像或空间纵深 |
| 仰拍定镜 | 机位低于人物视线30-45度，用于塑造威权人物 |

**禁用运镜方式**：急推急拉、手持晃动、大幅甩镜（破坏仪式感）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 压抑沉寂 | 长定镜+缓慢横移，人物沉默不语，烛火摇曳 |
| 承 | 暗流涌动 | 缓推近景，眼神特写，手部动作特写 |
| 转 | 爆发对峙 | 急推近景→特写，台词节奏加快，运镜转为短横移跟拍 |
| 合 | 沉寂余韵 | 缓拉至全景，人物定格，配乐淡出 |

---

## 4. 色彩特征

| 要素 | 描述 |
|---|---|
| 主色调 | 沉郁暗调（深褐、墨黑、青灰） |
| 点缀色 | 朱红（官袍、印信）、明黄（圣旨、帷幔） |
| 饱和度 | 中低饱和（30-50%） |
| 对比度 | 高对比（明暗差3-4档） |

---

## 5. 构图特征

| 要素 | 描述 |
|---|---|
| 构图方式 | 中正对称为主，关键转折用倾斜构图打破平衡 |
| 留白习惯 | 上方留白30%（突出顶光压迫），人物多置于画面下三分 |
| 前景遮挡 | 常用柱子、帷幔、屏风做前景遮挡，营造窥视感 |
| 纵深 | 大堂纵深长，机位多置于堂口向内拍，强调空间纵深 |

---

## 6. 转场偏好

| 转场方式 | 使用频率 | 适用场景 |
|---|---|---|
| 切 | 高频（70%） | 对话切换、镜头组接 |
| 叠化 | 中频（20%） | 时间流逝、场景过渡 |
| 闪回 | 低频（5%） | 关键回忆揭露 |
| 运镜转场 | 低频（5%） | 推向物体后切入下镜 |

---

## 7. 音效偏好

| 类型 | 内容 |
|---|---|
| 环境音 | 烛火噼啪、远处更鼓、风穿堂声 |
| 动效音 | 拍案重击、印信落地、衣物摩擦、佩剑碰撞 |
| 配乐风格 | 古琴/编钟为主，慢节奏，低频铺底 |

## 8. 典型镜头模板

### 模板 G01-T1 — 威权登场

```
镜头ID：场号-01
景别：全景 → 中景
运镜：镜头从堂口正中3米处中景，沿水平方向缓慢推进至人物前方1.5米处中近景
场景地点：S编号 县衙大堂内，青石板地面，烛火照明
环境光影：顶光自梁下垂直压下，色温2700K暖橙，高反差，烛火摇曳产生动态阴影
出场人物：C编号 角色名（堂上正中端坐，面朝堂下）
人物动作：C编号 缓缓抬手整理衣袖，目光自下而上扫过堂前
面部表情：C编号 眉眼平直，嘴角微压，下颌微抬（情绪：威权沉稳）
情绪基调：威仪登场
台词/音效：无台词 / 音效：烛火噼啪、远处更鼓声
转场方式：切
```

### 模板 G01-T2 — 拍案对峙

```
镜头ID：场号-05
景别：近景
运镜：镜头从人物肩后2米处沿水平方向匀速推进至0.5米近景
场景地点：S001 县衙大堂内，青石板地面，烛火照明
环境光影：顶光自梁下垂直压下，色温2700K暖橙，明暗对比强烈，烛火摇曳产生动态阴影
出场人物：C001 沈砚（案台后正中站立）；C002 李掌柜（案台前偏左，距案台1米）
人物动作：C001 右手猛地将账册拍在案台上（台词前0.5秒），身体微微前倾，左手按住案台边缘；C002 后退半步，双手在身前微微发抖
面部表情：C001 眉头紧锁，眼神锐利，嘴角紧绷，下颌发力（情绪：压抑的怒火）；C002 脸色刷白，瞳孔微缩，上下唇分离颤抖（情绪：惊恐）
情绪基调：压迫对峙
台词/音效：C001："这三千两银子，你作何解释？"（语气冰冷，重音在"解释"） / 音效：账册拍案重击声、烛火噼啪声
转场方式：切
```

### 模板 G01-T3 — 暗流揭示

```

人物动作：C001 右手拇指缓缓翻开账册第二页，指节因用力微微泛白
面部表情：C001 视线下垂注视账册，眉头微蹙，嘴角紧抿（情绪：审视思索）
情绪基调：暗流揭示
台词/音效：无台词 / 音效：纸张翻动声、烛火噼啪声、远处更鼓声
转场方式：切
```
