# G13 大漠武侠

> 大漠武侠题材。烈日黄沙+篝火暖红、广角大景+腾挪跟拍，营造孤傲决斗苍凉的氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 烈日顶光+篝火点光源，多向硬光 |
| 色温 | 白天5500K日光+夜晚2000K篝火暖红 |
| 明暗氛围 | 白天高调硬光，夜晚低调暖光 |
| 动态光影效果 | 黄沙飞舞、篝火摇曳、剑光闪烁 |

**辅助光源**：月光（夜晚）、刀剑反光、远处烽火台火光

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 广角大景 | 16-24mm广角，展示大漠辽阔 |
| 腾挪跟拍 | 与侠客腾挪动作同步跟拍，速度1.5-2米/秒 |
| 升格特写 | 升格4倍速下，剑光/血溅/侠客倒地特写 |
| 低角度仰拍 | 机位贴近沙地30度仰拍，塑造侠客孤傲感 |

**禁用运镜方式**：手持晃动（破坏武侠美感）、明调高光过曝（破坏苍凉感）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 孤傲出场 | 广角大景、侠客单骑入大漠、配乐苍凉 |
| 承 | 决斗剑光 | 腾挪跟拍、升格特写、剑光闪烁 |
| 转 | 苍凉落幕 | 缓拉至大景、侠客背影远去、配乐渐弱 |

---

## 4. 色彩特征

| 要素 | 描述 |
|---|---|
| 主色调 | 黄沙土黄+暗红血色+冷蓝夜空 |
| 点缀色 | 侠客衣袍的青黑/酒红、刀剑银白反光 |
| 饱和度 | 中饱和主体（40-55%）+黄沙高饱和 |
| 对比度 | 中高对比（明暗差3-4档） |

---

## 5. 构图特征

| 要素 | 描述 |
|---|---|
| 构图方式 | 大场景留白+人物剪影 |
| 留白习惯 | 上方天空留白50%，人物占画面1/8 |
| 前景遮挡 | 飞舞黄沙、枯草、破损旗帜做前景 |
| 纵深 | 深景深展示大漠纵深，远处沙丘清晰可见 |

## 8. 典型镜头模板

### 模板 G13-T1 — 孤傲出场

```
镜头ID：场号-01
景别：大远景 → 中景
运镜：镜头机位固定于沙丘斜侧方5米处，焦段自24mm广角匀速变至85mm中焦，从大远景压缩到中景，时长8秒
场景地点：S编号 大漠沙丘外，黄沙地面，烈日顶光照明
环境光影：烈日顶光自上方垂直压下色温5500K日光白，极高反差，黄沙飞舞产生动态光斑，远处沙丘因热浪扭曲
出场人物：C001 侠客（沙丘顶端正立，背对镜头，手中握长剑）；C002 坐骑黑马（沙丘下方伫立）
人物动作：C001 立于沙丘顶端纹丝不动，衣袍被大漠风吹得猎猎作响；右手握长剑剑尖垂地；C002 黑马在下方伫立，鬃毛随风飘动
面部表情：不可见（背影）
情绪基调：孤傲出场
台词/音效：无台词 / 音效：风沙呼啸、远处马嘶声、配乐二胡主旋律苍凉渐起
转场方式：切
```

### 模板 G13-T2 — 决斗剑光

```
镜头ID：场号-05
景别：中景 → 特写
运镜：镜头以C001为圆心，半径2米，围绕C001匀速环绕180度，升格4倍速，从中景过渡到特写，时长6秒（实际1.5秒）
场景地点：S编号 大漠决斗场外，黄沙地面，烈日顶光照明
环境光影：烈日顶光自上方垂直压下色温5500K日光白，极高反差，剑刃反光在空中划出银白光痕，黄沙飞舞产生动态光斑
出场人物：C001 侠客（决斗场中央正立，右手握长剑）；C003 反派刀客（C001对侧5米处，双手握大刀）
人物动作：C001 右手骤然挥剑，剑身化作银白光痕直斩C003颈侧；C003 双手举大刀格挡，刀剑相碰迸发火星
面部表情：C001 眉头紧锁，眼神锐利如刀，下颌发力（情绪：决斗专注）；C003 眉头深锁，嘴角扭曲上扬（情绪：狂暴对决）
情绪基调：决斗剑光
台词/音效：C001："受死！"（语气凌厉爆发） / 音效：刀剑相碰叮当炸响、黄沙飞舞沙沙声、配乐大鼓重音炸响
转场方式：白闪
```

### 模板 G13-T3 — 苍凉落幕

```
镜头ID：场号-12
景别：中景 → 大远景
运镜：镜头机位固定于侠客侧后方3米处，焦段自85mm中焦匀速变至24mm广角，从中景拉远至大远景，时长8秒
场景地点：S编号 大漠沙丘外，黄沙地面，黄昏余晖+篝火暖红照明
环境光影：黄昏余晖自西方平射色温2500K暖橙，远处篝火余烬散光色温2000K暖红，极高反差，黄昏余晖在沙丘上拉长阴影
出场人物：C001 侠客（沙丘斜侧方正立，背对镜头，右手握长剑剑尖垂地）；C002 坐骑黑马（沙丘下方伫立）
人物动作：C001 立于沙丘斜侧方，衣袍被大漠风吹得猎猎作响；右手握长剑剑尖垂地，剑刃上残留血迹；缓步走向C002
面部表情：不可见（背影）
情绪基调：苍凉落幕
台词/音效：无台词 / 音效：风沙呼啸渐弱、远处马嘶声、配乐二胡主旋律苍凉渐弱收尾
转场方式：叠化
```
