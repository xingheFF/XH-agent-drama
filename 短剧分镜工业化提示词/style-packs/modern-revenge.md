# G02 现代复仇

> 现代都市背景下的复仇爽文题材。冷蓝霓虹、手持晃动、急推镜头，营造压抑到爆发的爽感节奏。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 霓虹灯+路灯+车灯混合，多向光 |
| 色温 | 主体冷蓝6000K+点缀霓虹紫粉2500K |
| 明暗氛围 | 高反差低调光，霓虹光斑跳跃 |
| 动态光影效果 | 车灯扫过、霓虹闪烁、雨水反光 |

**辅助光源**：手机/电脑屏幕蓝光、烟头红点、玻璃幕墙反射光

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 手持晃动 | 紧张对峙场景，幅度0.1-0.3米，频率1-3Hz |
| 急推 | 反转时刻，从2米中景急推至0.3米特写，>2米/秒 |
| 跟拍侧前方 | 与人物同步前行，保持1-1.5米距离 |
| 倾斜构图 | 仰拍+荷兰角，塑造主角威权 |

**禁用运镜方式**：缓慢匀速横移、过长的固定镜头（破坏爽感节奏）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 隐忍压抑 | 主角低头特写、烟头明灭、背景虚化 |
| 承 | 反转登场 | 急推主角脸部、仰拍构图、配乐炸裂 |
| 转 | 碾压压制 | 跟拍主角前行、对手后退跟拍、手持晃动 |
| 合 | 沉稳离场 | 背影长焦压缩、车灯拖尾、慢门效果 |

---

## 4. 色彩特征

| 要素 | 描述 |
|---|---|
| 主色调 | 冷蓝霓虹+暗灰金属 |
| 点缀色 | 霓虹紫粉、血液暗红、烟头橙红 |
| 饱和度 | 中高饱和霓虹部分，主体低饱和（20-40%） |
| 对比度 | 极高对比（明暗差4-5档） |

---

## 5. 构图特征

| 要素 | 描述 |
|---|---|
| 构图方式 | 倾斜构图（荷兰角）打破平衡，主角居中威权 |
| 留白习惯 | 大量负空间，主体常占画面1/3 |
| 前景遮挡 | 玻璃、百叶窗、雨幕做前景遮挡，营造窥视感 |
| 纵深 | 长焦压缩背景，浅景深虚化，主体凸显 |

---

## 6. 转场偏好

| 转场方式 | 使用频率 | 适用场景 |
|---|---|---|
| 切 | 高频（80%） | 节奏紧凑，爽感推进 |
| 闪回 | 低频（10%） | 主角回忆受辱场景 |
| 白闪 | 低频（5%） | 反转爆发瞬间 |
| 运镜转场 | 低频（5%） | 甩镜衔接下镜 |

---

## 7. 音效偏好

| 类型 | 内容 |
|---|---|
| 环境音 | 雨声、车流声、远处警笛、霓虹灯电流声 |
| 动效音 | 高跟鞋敲击、皮鞋踏地、打火机点燃、玻璃杯碰撞 |
| 配乐风格 | 电子鼓点+低频贝斯，反转时刻重音炸裂 |

## 8. 典型镜头模板

### 模板 G02-T1 — 隐忍登场

```
镜头ID：场号-01
景别：特写
运镜：镜头自主角低垂眼睑上方0.2米处垂直匀速下降至0.05米处特写，速度0.1米/秒
场景地点：S编号 雨夜街角外，湿滑石板路，霓虹灯+路灯照明
环境光影：霓虹紫粉光自左上方斜射，色温混合2500K+6000K，高反差，雨水在地面反光闪烁
出场人物：C编号 主角（街角站立，撑黑色长柄伞，伞檐压低遮住眉眼）
人物动作：C编号 右手持伞微微下压，左手插在风衣口袋中，烟头明灭可见
面部表情：C编号 眉眼被伞檐阴影遮蔽，仅可见下颌线条紧绷，嘴角微压（情绪：隐忍压抑）
情绪基调：隐忍登场
台词/音效：无台词 / 音效：雨声密集、远处车流声、打火机点燃轻响
转场方式：切
```

### 模板 G02-T2 — 反转急推

```
镜头ID：场号-05
景别：中景 → 特写
运镜：镜头从主角正前方2米处中景，沿水平方向急推至0.3米处特写，速度>2米/秒，时长0.5秒
场景地点：S编号 高端会所大堂内，大理石地面，水晶吊灯照明
环境光影：水晶吊灯顶光自上而下，色温4000K暖白，高反差，吊灯棱镜在地面投射彩虹光斑
出场人物：C编号 主角（大堂正中站立，背对镜头）；C编号 对手（前方5米处，面朝主角）
人物动作：C编号 主角骤然转身，风衣下摆扬起弧线；C编号 对手瞳孔骤缩，后退半步撞上立柱
面部表情：C编号 主角眉眼平直，嘴角微挑，眼神冰冷如刀（情绪：碾压压制）；C编号 对手瞳孔骤缩，嘴唇开合两次失声（情绪：震惊失声）
情绪基调：反转碾压
台词/音效：C编号 主角："三年了，你终于知道怕了？"（语气平缓，重音在"怕"） / 音效：风衣布料摩擦声、配乐重音炸裂
转场方式：白闪
```

### 模板 G02-T3 — 跟拍碾压

```
镜头ID：场号-08
景别：中景
运镜：镜头在主角侧前方1米处，与主角保持同步匀速前行（速度约0.8米/秒），从中景维持到中景
场景地点：S编号 公司长廊内，木地板地面，日光灯管照明
环境光影：日光灯管顶光自上而下，色温6500K冷白，高反差，灯管在长廊纵深形成明暗交替
出场人物：C编号 主角（长廊正中大步行走，速度0.8米/秒）；多名下属（长廊两侧低头避让）
人物动作：C编号 主角双手插裤袋大步前行，每一步皮鞋踏地声清晰，余光扫过两侧下属
面部表情：C编号 主角眉眼平直，嘴角微压，下颌微抬（情绪：威权碾压）
情绪基调：碾压推进
台词/音效：无台词 / 音效：皮鞋踏地声节奏稳定、下属衣料摩擦声、配乐低频推进
转场方式：切
```

### 模板 G02-T4 — 离场长焦

```
镜头ID：场号-12
景别：中景
运镜：镜头机位固定于主角侧后方3米处，焦段自50mm匀速变至200mm长焦，从中景压缩到近景，时长4秒
场景地点：S编号 雨夜街角外，湿滑石板路，霓虹灯+路灯照明
环境光影：霓虹紫粉光自左上方斜射，色温混合2500K+6000K，高反差，雨水在主角背影上反光
出场人物：C编号 主角（街角背影大步离去，撑黑色长柄伞）
人物动作：C编号 主角大步走入雨幕，背影逐渐被长焦压缩放大，伞檐雨水滴落
面部表情：不可见（背影）
情绪基调：沉稳离场
台词/音效：无台词 / 音效：雨声渐密、皮鞋踏地声渐远、配乐低频收尾
转场方式：叠化
```
