# G05 都市职场

> 都市职场题材。日光白+冷灰办公光、跟拍越轴对切，营造紧绷博弈胜出的现代感。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 办公室日光灯管+落地窗自然光 |
| 色温 | 5500-6500K 冷白+灰蓝 |
| 明暗氛围 | 中调偏亮，高反差于玻璃幕墙反光处 |
| 动态光影效果 | 百叶窗条纹光投影、云影移动产生明暗变化 |

**辅助光源**：电脑屏幕蓝光、咖啡机暖黄点光源

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 跟拍前侧 | 与人物同步前行，保持1米距离，速度0.8-1米/秒 |
| 越轴对切 | 双人对话机位越过轴线切换，营造紧张感 |
| 缓推近景 | 从1.5米中近景缓推至0.5米近景，速度0.4米/秒 |
| 长焦固定 | 85-135mm长焦机位固定，浅景深突出主体 |

**禁用运镜方式**：手持晃动（除非冲突爆发）、急推急拉（破坏职场理性感）、环绕镜头（过于浪漫）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 紧绷开场 | 跟拍前行、长焦固定展示办公室纵深 |
| 承 | 博弈对峙 | 越轴对切双人对话、缓推近景 |
| 转 | 胜出反转 | 急推特写、白闪、对手失态特写 |
| 合 | 沉稳收尾 | 缓拉至全景、主角背影长焦压缩 |

## 8. 典型镜头模板

### 模板 G05-T1 — 跟拍入场

```
镜头ID：场号-01
景别：中景
运镜：镜头在主角侧前方1米处，与主角保持同步匀速前行（速度约0.8米/秒），从中景维持到中景，时长5秒
场景地点：S编号 公司长廊内，木地板地面，日光灯管照明
环境光影：日光灯管自上方平射，色温6500K冷白，中调偏亮，灯管在长廊纵深形成明暗交替
出场人物：C001 主角（长廊正中大步行走）；多名下属（长廊两侧低头避让）
人物动作：C001 双手插西装裤袋大步前行，每一步皮鞋踏地声清晰，余光扫过两侧下属
面部表情：C001 眉眼平直，嘴角微压，下颌微抬（情绪：威权碾压）
情绪基调：紧绷开场
台词/音效：无台词 / 音效：皮鞋踏地声节奏稳定、键盘敲击声远处传来、配乐电子节拍渐起
转场方式：切
```

### 模板 G05-T2 — 越轴博弈

```
镜头ID：场号-05
景别：近景
运镜：镜头机位越过轴线在双人侧方1米处切换，从中景A过渡到中景B，每镜2秒，速度0.5米/秒
场景地点：S编号 CEO办公室内，深色木地板，落地窗日光照明
环境光影：落地窗光自左方柔射，色温5500K冷白，中调偏亮，玻璃幕墙外云影移动产生明暗变化
出场人物：C001 主角（办公桌前正坐）；C002 对手（办公桌对侧正坐）
人物动作：C001 双手交叠置于桌上，身体微微前倾；C002 后背紧贴椅背，右手食指轻叩桌面
面部表情：C001 眉眼平直，嘴角微扬，眼神沉稳（情绪：掌控博弈）；C002 眉头紧锁，嘴角下压，眼神闪烁（情绪：被动招架）
情绪基调：博弈对峙
台词/音效：C001："这个项目，没有你，我会做得更好。"（语气平缓坚定） / 音效：签字笔按下咔嗒声、远处电梯运行声、配乐电子节拍渐强
转场方式：切
```

### 模板 G05-T3 — 胜出反转

```
镜头ID：场号-09
景别：中景 → 特写
运镜：镜头从主角正前方2米处中景，沿水平方向急推至0.3米处特写，速度>2米/秒，时长0.5秒
场景地点：S编号 会议室内部，灰色地毯地面，投影仪+日光灯照明
环境光影：投影仪蓝光自前方平射，色温混合冷白+蓝紫，中调偏亮，投影在主角脸上形成光斑
出场人物：C001 主角（会议桌前端站立，面朝众人）；多名董事（会议桌两侧坐立，面朝主角）
人物动作：C001 双手撑在会议桌前端，身体前倾15度；多名董事面面相觑，其中一人缓缓站起开始鼓掌
面部表情：C001 眉眼平直，嘴角微扬，下颌微抬（情绪：胜出掌控）；董事们表情从震惊转为钦佩
情绪基调：胜出反转
台词/音效：C001："这是最后的方案，通过或否决，请表态。"（语气沉稳有力） / 音效：掌声渐起、纸张翻动声密集、配乐电子节拍重音炸响
转场方式：白闪
```

### 模板 G05-T4 — 沉稳收尾

```
镜头ID：场号-12
景别：中景
运镜：镜头机位固定于主角侧后方3米处，焦段自50mm匀速变至135mm长焦，从中景压缩到近景，时长4秒
场景地点：S编号 公司落地窗前内，木地板地面，日落照明
环境光影：日落金光自落地窗外平射，色温2500K暖橙，中调偏暖，玻璃幕墙反射夕阳余晖
出场人物：C001 主角（落地窗前背对镜头站立，双手插裤袋）
人物动作：C001 双手插西装裤袋，背对镜头凝视窗外城市天际线，肩膀线条沉稳
面部表情：不可见（背影）
情绪基调：沉稳收尾
台词/音效：无台词 / 音效：远处城市车流声、配乐电子节拍渐弱
转场方式：叠化
```
