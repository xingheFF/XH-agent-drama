# G06 民国谍战

> 民国谍战题材。雨夜冷蓝+车灯硬光、长焦压缩+甩镜，营造危机智斗牺牲的紧张氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 路灯+车灯+煤油灯，多向硬光 |
| 色温 | 主体冷蓝6500K+点缀钠灯黄3000K |
| 明暗氛围 | 极低调光，阴影占70%以上 |
| 动态光影效果 | 雨幕散光、车灯扫过、煤油灯火焰跳动 |

**辅助光源**：火柴光、烟头红点、远处霓虹

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 长焦压缩 | 135-200mm长焦，主体与背景压缩贴近 |
| 甩镜衔接 | 主体A急速甩向主体B，速度>3米/秒 |
| 慢推低调 | 从3米中景缓推至0.5米近景，速度0.3米/秒 |
| 主观窥视 | 主观视角越过门框/窗框/百叶窗缓慢推进 |

**禁用运镜方式**：明调高光、对称稳定构图、欢快跳切

---

## 3. 情绪节奏

| 阶段 | 情绪 | 典型镜头 |

| 转 | 牺牲反转 | 急推特写、白闪/黑场切换、枪声留白 |
| 合 | 沉寂悲壮 | 缓拉至全景、人物定格、配乐未止 |

## 8. 典型镜头模板

### 模板 G06-T1 — 危机暗涌

```
镜头ID：场号-01
景别：全景
运镜：镜头机位固定于街角2米处，水平方向自左向右匀速旋转45度，速度0.2米/秒，从全景过渡到中景
场景地点：S编号 民国雨夜街道外，湿滑石板路，路灯+煤油灯照明
环境光影：路灯自上方垂直压下色温3000K暖黄，煤油灯自街边小店门口散光色温2000K暖橙，极低调光，雨幕散光柔和
出场人物：C001 谍战特工（街角撑黑伞站立）；C002 接头人（街对面路灯下站立）
人物动作：C001 撑伞立于街角阴影中，右手插风衣口袋握枪；C002 在街对面路灯下站定，左手三次轻拍大衣口袋
面部表情：C001 眉头微蹙，眼神锐利如刀，下颌线条紧绷（情绪：警惕暗涌）；C002 眉眼平直，嘴角微压（情绪：冷静接头）
情绪基调：危机暗涌
台词/音效：无台词 / 音效：雨声密集、远处警笛声、煤油灯火焰噼啪声
转场方式：切
```
