# G09 家庭伦理

> 家庭伦理题材。暖黄居家光+晨昏柔光、固定长镜+缓慢横移，营造温情裂痕和解的氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 居家吊灯+台灯+窗光，多向柔光 |
| 色温 | 2700-3500K 暖黄为主 |
| 明暗氛围 | 中调偏暖，无明显强阴影 |
| 动态光影效果 | 窗帘透光晃动、晨光渐变、晚餐烛光摇曳 |

**辅助光源**：电视屏幕蓝光、餐桌上方的暖黄吊灯

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 固定长镜 | 机位完全固定，时长5-10秒，留白让观众感受情绪 |
| 缓慢横移 | 沿水平方向0.2-0.3米/秒横移，展示家庭空间纵深 |
| 缓推近景 | 从1.5米中近景缓推至0.5米近景，速度0.3米/秒 |
| 缓拉全景 | 从中景缓拉至全景展示家庭群像 |

**禁用运镜方式**：急推急拉（破坏温情感）、手持晃动（破坏稳定感）、长焦压缩（过于疏离）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 温情日常 | 固定长镜、缓慢横移、家庭聚餐场景 |
| 承 | 裂痕显现 | 缓推近景、特写情绪、双人越轴对切 |
| 转 | 和解拥抱 | 缓拉全景、双人对称构图、暖光高调 |
| 合 | 生活继续 | 固定长镜、人物定格、配乐渐弱 |

---

## 4. 色彩特征

| 要素 | 描述 |
|---|---|
| 主色调 | 暖黄米白+旧物棕褐 |
| 点缀色 | 餐桌绿植、窗帘浅蓝、相框金色 |
| 饱和度 | 中饱和（40-55%） |
| 对比度 | 低对比（明暗差1-2档），高调偏暖 |

---

## 5. 构图特征

| 要素 | 描述 |
|---|---|
| 构图方式 | 居家空间+人物居中 |
| 留白习惯 | 上方留白20%，人物居中偏下 |
| 前景遮挡 | 餐具、花瓶、相框做前景 |
| 纵深 | 中景深展示家庭空间纵深 |

---

## 6. 转场偏好

| 转场方式 | 使用频率 | 适用场景 |
|---|---|---|
| 切 | 中频（50%） | 节奏适中 |
| 叠化 | 高频（40%） | 时间流逝、日常过渡 |
| 闪回 | 低频（5%） | 关键回忆 |
| 黑场 | 低频（5%） | 段落留白 |

---

## 7. 音效偏好

| 类型 | 内容 |
|---|---|
| 环境音 | 厨房炒菜声、电视节目声、钟表滴答、窗外孩童嬉闹 |
| 动效音 | 筷子碰碗声、椅子挪动声、纸张翻动声 |
| 配乐风格 | 钢琴+小提琴+轻电子，节奏舒缓 |

```

景别：全景
```
