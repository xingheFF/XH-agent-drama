# G03 甜宠

> 甜宠恋爱题材。柔光粉调、过曝高调、环绕镜头与特写跳切，营造轻快心动治愈氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 柔光窗光+补光灯，多向柔光 |
| 色温 | 5500K 日光白偏暖+局部4500K暖黄 |
| 明暗氛围 | 高调明亮，无明显阴影 |
| 动态光影效果 | 窗帘透光晃动、逆光金边、过曝高光溢出 |

**辅助光源**：反光板补面部光（强度等同主光）、节日彩灯点光源

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 环绕 | 以双人中心为圆心，半径1.5米环绕180-360度，速度0.5米/秒 |
| 特写跳切 | 双人近景与特写快速切换，每镜0.8-1.5秒 |
| 缓推近景 | 从1.5米中近景缓推至0.5米近景，速度0.3米/秒 |
| 跟拍前侧 | 与人物同步前行，保持1米距离，速度0.8米/秒 |

**禁用运镜方式**：手持晃动（破坏甜蜜感）、急推急拉（破坏柔和感）、长焦压缩（过于疏离）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 轻快相遇 | 跟拍前行、跳切双人特写、自然光下高调明亮 |
| 承 | 心动瞬间 | 缓推近景、环绕镜头、逆光金边特写 |
| 转 | 甜蜜互动 | 双人对称构图、特写跳切、对视0.5秒 |
| 合 | 治愈余韵 | 双人背影缓拉、柔光过曝、配乐渐弱 |

---

## 4. 色彩特征

| 要素 | 描述 |
|---|---|
| 主色调 | 粉白蓝+高饱和暖色 |
| 点缀色 | 樱花粉、莫兰迪蓝、奶油黄 |
| 饱和度 | 中高饱和（60-75%） |
| 对比度 | 低对比（明暗差1-2档），高调为主 |

---

## 5. 构图特征

| 要素 | 描述 |
|---|---|
| 构图方式 | 双人对称+中心构图 |
| 留白习惯 | 上方留白20%，人物居中偏上 |
| 前景遮挡 | 樱花花瓣、气球、纱帘轻柔前景 |
| 纵深 | 浅景深虚化，背景柔焦散景 |

---

## 6. 转场偏好

## 8. 典型镜头模板

### 模板 G03-T1 — 心动对视

```
镜头ID：场号-04
景别：近景 → 特写
运镜：镜头从双人正前方1.5米处中近景，沿水平方向缓推至0.3米处双人特写，速度0.3米/秒，时长4秒
场景地点：S编号 樱花树下公园外，草坪地面，日光照明
环境光影：自然日光自上方斜射，色温5500K，高调明亮，逆光金边勾勒人物轮廓，樱花花瓣飘落形成动态光影
出场人物：C001 主角A（草坪左侧站立，距主角B0.5米）；C002 主角B（草坪右侧站立，面朝A）
人物动作：C001 缓缓抬起右手欲触碰C002脸颊，在距脸颊0.05米处停顿2秒；C002 微微仰头，睫毛轻颤
面部表情：C001 眉眼柔和，嘴角微扬，眼神温柔（情绪：心动）；C002 眼睑微垂，睫毛轻颤，嘴角轻抿（情绪：害羞心动）
情绪基调：心动瞬间
台词/音效：无台词 / 音效：风铃声轻响、远处鸟鸣、配乐钢琴主旋律渐起
转场方式：白闪
```

### 模板 G03-T2 — 环绕告白

```
镜头ID：场号-07
景别：中景
运镜：镜头以双人中心为圆心，半径1.5米，围绕双人匀速环绕180度，速度0.5米/秒，从中景过渡到中近景
场景地点：S编号 咖啡厅落地窗旁内，木地板地面，窗光照明
环境光影：窗光自左方柔射，色温5500K偏暖，高调明亮，咖啡厅暖黄灯点缀，窗帘透光晃动
出场人物：C001 主角A（落地窗前正对C002）；C002 主角B（落地窗前正对C001，距0.8米）
人物动作：C001 双手握住C002的右手，身体微微前倾；C002 垂眸注视两人交握的手，左手无意识攥紧衣角
面部表情：C001 眉头微舒，眼神专注坚定（情绪：认真告白）；C002 睫毛微颤，嘴角欲语还休（情绪：犹豫心动）
情绪基调：告白心动
台词/音效：C001："我喜欢你，从第一眼开始。"（语气温柔坚定） / 音效：咖啡机运作声、远处轻音乐、衣物柔软摩擦声
转场方式：切
```

### 模板 G03-T3 — 治愈余韵

