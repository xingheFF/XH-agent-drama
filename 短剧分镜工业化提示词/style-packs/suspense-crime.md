# G04 悬疑罪案

> 悬疑罪案题材。低调绿黄混色光、慢摇窥视视角、大量负空间，营造诡异揭露战栗氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 局部光源（台灯/路灯/月光），单一方向 |
| 色温 | 主体绿黄混色3000-4000K，月光冷白6500K |
| 明暗氛围 | 极低调光，暗部占画面70%以上 |
| 动态光影效果 | 雨水反光闪烁、车灯扫过、霓虹灯电流声伴随的光颤 |

**辅助光源**：手电光束、烟头红点、远处车灯

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 慢摇窥视 | 机位固定，水平方向自左向右匀速旋转30-60度，速度0.2米/秒 |
| 缓推低调 | 从3米全景缓推至0.5米近景，速度0.3米/秒 |
| 主观窥视 | 主观视角越过门框/窗框/百叶窗缓慢推进 |
| 长焦固定 | 200mm长焦机位固定，浅景深突出主体 |

**禁用运镜方式**：快速跟拍（破坏悬疑感）、明调高光（破坏低调感）、对称构图（过于稳定）

---

## 3. 情绪节奏

三段式情绪曲线：

| 阶段 | 情绪 | 典型镜头 |
|---|---|---|
| 起 | 诡异铺垫 | 慢摇空镜、低调光下细节特写、环境音密集 |
| 承 | 揭露线索 | 缓推近景、手部动作特写、关键道具特写 |
| 转 | 战栗反转 | 急推特写、手持晃动、白闪/黑场切换 |
| 合 | 余韵未消 | 缓拉至全景、人物定格、配乐未止 |

---

## 4. 色彩特征

| 要素 | 描述 |
|---|---|
| 主色调 | 灰绿冷调+局部血红 |
| 点缀色 | 钠灯黄、铁锈红、霉斑绿 |
| 饱和度 | 低饱和（20-35%） |
| 对比度 | 极高对比（明暗差5档） |

---

## 5. 构图特征

| 要素 | 描述 |
|---|---|
| 构图方式 | 不对称+大量负空间 |
| 留白习惯 | 主体常占画面1/4，周围大块阴影 |
| 前景遮挡 | 门框、窗框、百叶窗、雨幕、烟雾做前景 |
| 纵深 | 深景深展示空间纵深，营造"角落里有人"的感觉 |

---

## 6. 转场偏好

| 转场方式 | 使用频率 | 适用场景 |
|---|---|---|
| 切 | 高频（60%） | 节奏推进 |
| 黑场 | 中频（20%） | 段落留白、窒息感 |
| 闪回 | 中频（15%） | 受害者回忆、线索回溯 |
| 叠化 | 低频（5%） | 时间流逝 |

---

## 7. 音效偏好

| 类型 | 内容 |
|---|---|
| 环境音 | 滴水声、电流嗡鸣、远处犬吠、风穿过空房 |
| 动效音 | 脚步回响、门轴吱呀、纸张翻动、玻璃碎裂 |
| 配乐风格 | 弦乐不和谐音+低频持续音+突然静默 |

---

## 8. 典型镜头模板

### 模板 G04-T1 — 诡异空镜

```
镜头ID：场号-01
景别：全景
运镜：镜头机位固定于房间门口1.5米处，水平方向自左向右匀速旋转45度，速度0.2米/秒，从全景过渡到中景
场景地点：S编号 废弃厂房内，水泥地面，月光+路灯照明
环境光影：月光自破损窗户斜射，色温6500K冷白，钠灯黄光自远处透入3000K，极低调光，月光在地上形成矩形光斑
出场人物：无
人物动作：无
面部表情：无
情绪基调：诡异铺垫
台词/音效：无台词 / 音效：滴水声节奏稳定、电流嗡鸣低频、远处犬吠一声
转场方式：切
```

```

面部表情：不可见（背影）
情绪基调：揭露线索
台词/音效：C001："这一天，所有证人都不在场。"（语气低沉平缓） / 音效：纸张翻动声、台灯电流嗡鸣、远处钟摆声
转场方式：切
```

### 模板 G04-T3 — 战栗反转

```
镜头ID：场号-09
景别：中景 → 特写
运镜：镜头从主体正前方2米处中景，沿水平方向急推至0.3米处特写，速度>2米/秒，时长0.5秒
场景地点：S编号 停尸房内，瓷砖地面，日光灯照明
环境光影：日光灯自上方平射，色温6500K冷白，极低调光，灯管在尸体上投射惨白光，暗部占画面75%
出场人物：C001 侦探（尸体侧方1米，面朝尸体）；C002 法医（尸体对侧，面朝C001）
人物动作：C001 双手骤然攥紧尸体衣领，身体前倾；C002 后退半步，手中的解剖刀掉落在瓷砖地面
面部表情：C001 眉头紧锁，瞳孔骤缩，下颌发力（情绪：震惊战栗）；C002 嘴角下垂，眼神闪烁（情绪：心虚暴露）
情绪基调：战栗反转
台词/音效：C002："不、不是我……"（语气颤抖结巴） / 音效：解剖刀落地金属撞击声、心跳声急促、配乐弦乐不和谐音炸响
转场方式：黑场
```

### 模板 G04-T4 — 余韵窥视

```

环境光影：月光自走廊尽头窗户平射，色温6500K冷白，极低调光，门缝在地面投射细长光带

人物动作：C003 缓缓转身，露出半张被阴影遮蔽的脸，嘴角微扬
面部表情：C003 眉眼被帽檐阴影遮蔽，仅可见嘴角缓缓上扬（情绪：诡异的满足）
情绪基调：余韵未消
台词/音效：无台词 / 音效：风穿过旧宅的呜咽声、远处钟敲十二响、配乐低频持续音未止

```
