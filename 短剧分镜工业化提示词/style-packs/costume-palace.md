# G14 宫廷古装

> 宫廷古装题材。金碧辉煌+琉璃冷光、俯仰宫廷+对称威仪，营造华丽权谋悲歌的氛围。

---

## 1. 光影基调

| 参数 | 描述 |
|---|---|
| 主光源 | 宫灯+琉璃天窗光+烛火，多向暖光 |
| 色温 | 主体2700-3500K暖橙+琉璃冷光6000K |
| 明暗氛围 | 中调偏暖，宫灯高调，柱廊低调 |
| 动态光影效果 | 宫灯摇曳、琉璃天窗光柱穿透、烛火闪烁 |

**辅助光源**：金箔反光、丝绸反光、远处香炉火光

---

## 2. 运镜偏好

| 常用运动 | 描述 |
|---|---|
| 俯仰宫廷 | 仰拍宫殿穹顶/俯拍宫道纵深，展示宫廷威仪 |
| 对称缓推 | 机位居中正对人物，从中景对称缓推至近景 |
| 缓慢横移 | 沿水平方向0.2-0.3米/秒横移，展示宫道纵深 |
| 长焦固定 | 135mm长焦机位固定，浅景深突出主体 |

**禁用运镜方式**：手持晃动（破坏宫廷威仪感）、急推急拉（破坏庄重感）

## 8. 典型镜头模板

### 模板 G14-T1 — 华丽登场

```
镜头ID：场号-01
景别：中景 → 近景
运镜：镜头机位居中正对人物，从2米处中景沿水平方向对称缓推至0.8米处近景，速度0.3米/秒，时长6秒
场景地点：S编号 宫廷大殿内，金砖地面，宫灯+琉璃天窗光照明
环境光影：宫灯自四周散光色温2700K暖橙，琉璃天窗光自上方垂直压下色温6000K冷白，中调偏暖，宫灯高调，柱廊低调，琉璃光柱穿透
出场人物：C001 帝王（大殿中央正立，背对镜头，身着明黄龙袍）；多名宫女（大殿两侧垂首侍立）
人物动作：C001 立于大殿中央纹丝不动，明黄龙袍衣摆拖地，右手缓缓抬起整理腰间玉带；多名宫女垂首侍立纹丝不动
面部表情：不可见（背影）
情绪基调：华丽登场
台词/音效：无台词 / 音效：宫灯摇曳轻响、远处编钟一声、配乐编钟主旋律庄严渐起
转场方式：切
```

### 模板 G14-T2 — 权谋博弈

```
镜头ID：场号-05
景别：近景
运镜：镜头机位居中正对双人，从2米处中近景沿水平方向对称缓推至0.8米处近景，速度0.3米/秒，时长5秒
场景地点：S编号 宫廷御书房内，金砖地面，宫灯+琉璃天窗光照明
环境光影：宫灯自书桌上方散光色温2700K暖橙，琉璃天窗光自上方垂直压下色温6000K冷白，中调偏暖，宫灯在书桌上投射暖黄光斑
出场人物：C001 帝王（御书桌后正坐，面朝C002）；C002 权臣（御书桌前正立，面朝C001）
人物动作：C001 右手执朱笔缓缓在奏折上批注，每写一字停顿一秒；C002 双手交叠于身前，身体微微前倾呈恭谦之态
面部表情：C001 眉眼平直，嘴角微压，下颌微抬（情绪：威权审视）；C002 眉头微舒，嘴角微扬，眼神中带算计（情绪：权谋恭谦）
情绪基调：权谋博弈
台词/音效：C002："陛下，此事还需从长计议。"（语气恭谦暗含锋芒） / 音效：朱笔批注沙沙声、宫灯摇曳轻响、配乐古琴低音渐起
转场方式：切
```

### 模板 G14-T3 — 悲歌落幕

```
镜头ID：场号-12
景别：中景 → 全景
运镜：镜头机位固定于宫道斜侧方3米处，焦段自50mm中焦匀速变至24mm广角，从中景拉远至全景，时长8秒
场景地点：S编号 宫廷长道内，金砖地面，宫灯+琉璃天窗光照明
环境光影：宫灯自长道两侧散光色温2700K暖橙，琉璃天窗光自上方垂直压下色温6000K冷白，中调偏暖，宫灯在长道两侧投射暖黄光斑
出场人物：C001 帝王（长道中央正立，背对镜头，明黄龙袍衣摆拖地）；多名宫女（长道两侧垂首侍立）
人物动作：C001 立于长道中央纹丝不动，右手缓缓抬起抚摸腰间玉带；凝视长道尽头渐远的宫门
面部表情：不可见（背影）
情绪基调：悲歌落幕
台词/音效：无台词 / 音效：宫灯摇曳轻响、远处编钟渐弱、配乐编钟主旋律庄严渐弱收尾
转场方式：叠化
```
